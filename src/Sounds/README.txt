This directory should contain OGG sound files for voice alerts.
Example mappings (SpellID -> File):
12294 -> MortalStrike.ogg
7384  -> Overpower.ogg
5308  -> Execute.ogg
772   -> Rend.ogg
1464  -> Slam.ogg
46924 -> Bladestorm.ogg

The Audio module will look for these files based on the SpellID embedded in the Profile.
