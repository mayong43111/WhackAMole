## 核心设计

**APL目标**：生存优先 → 稳定循环 → 怒气管理

**防御战士三大支柱**：

1. **生存能力**
   - 盾牌格挡：主动提升格挡100%，持续10秒
   - 破釜沉舟：增加30%最大生命值，持续20秒
   - 盾墙：60%伤害减免，持续12秒
   - 狂怒回复：治疗+吸收盾+免死（天赋增强）

2. **威胁控制**
   - 盾牌猛击：主要威胁技能，CD 6秒
   - 复仇：5怒气必爆，DPE最高
   - 毁灭打击：叠破甲，稳定威胁，4T1威胁+30%
   - 英勇打击：不占GCD，高威胁

3. **怒气管理**
   - 怒气来源：受伤、盾牌格挡、复仇触发
   - 优先低消耗技能（复仇5、毁灭15、盾猛20）
   - 怒气≥60立即使用英勇打击泄怒

**属性优先级**：
- 防御540（免爆）> 命中8% > 精准26 > 4T1套装
- 其余：耐力 > 闪避 > 招架 > 格挡值

## APL 配置

当前插件使用的SimC风格APL（通用防御战设计）：

```lua
-- === 生存优先（血量阈值触发）===

"actions+=/shield_wall,if=player.health_pct<20",
  -- 盾墙：血量<20%终极保命
  -- 理由：60%减伤持续12秒，确保不被秒杀
  
"actions+=/last_stand,if=player.health_pct<35",
  -- 破釜沉舟：血量<35%增加生命值上限
  -- 理由：+30%最大生命值持续20秒，给治疗争取时间

-- === 主动防御 ===

"actions+=/shield_block,if=cooldown.shield_block.ready",
  -- 盾牌格挡：CD好就用
  -- 理由：100%格挡持续10秒，大幅降低伤害，触发复仇机会
  
"actions+=/shield_bash",
  -- 盾牌反击：打断施法（需手动判断）
  -- 理由：降低魔法伤害，打断关键技能
  -- 注意：target.is_casting 当前不支持，需手动使用

-- === 威胁循环 ===

"actions+=/revenge",
  -- 复仇：最高威胁优先级
  -- 理由：5怒气必爆，DPE最高（威胁/怒气比是盾猛的3-4倍）
  -- 触发：格挡/招架/躲闪后可用
  
"actions+=/shield_slam",
  -- 盾牌猛击：主要威胁技能
  -- 理由：20怒气高威胁，CD 6秒，必须卡CD使用
  
"actions+=/thunder_clap,if=!debuff.thunder_clap.up",
  -- 雷霆一击：Debuff不存在时补上
  -- 理由：降低攻击速度10%，同时提供AOE威胁
  -- AOE场景：手动优先使用（配合复仇高频触发）
  
"actions+=/devastate",
  -- 毁灭打击：填充技能
  -- 理由：15怒气叠破甲（最多5层），4T1威胁+30%

-- === 怒气管理 ===

"actions+=/heroic_strike,if=player.power.rage.current>=60",
  -- 英勇打击：怒气≥60时泄怒
  -- 理由：不占GCD，将溢出怒气转化为威胁，防止浪费
  -- 重要：怒气上限100，超过会浪费

-- === Buff维持 ===

"actions+=/commanding_shout,if=buff.commanding_shout.remains<30",
  -- 命令怒吼：Buff剩余<30秒刷新
  -- 理由：增加团队生命值和耐力
  
"actions+=/demoralizing_shout,if=!debuff.demoralizing_shout.up",
  -- 挫志怒吼：Debuff消失时补上
  -- 理由：降低周围敌人攻击强度
```

**关键条件说明**：
- `player.health_pct<20/35`: 血量百分比阈值 ✅
- `cooldown.shield_block.ready`: CD就绪检测 ✅
- `target.is_casting`: 目标正在施法 ❌ **当前不支持**
- `!debuff.thunder_clap.up`: Debuff不存在 ✅
- `active_enemies>=2`: 多目标场景 ⚠️ **固定为1（占位符）**
- `player.power.rage.current>=60`: 怒气≥60 ✅

**设计原则**：
- 生存第一：血量阈值在所有输出之前
- 稳定循环：复仇>盾猛>毁灭，优先级清晰
- 怒气效率：≥60怒气立即泄怒
- 通用性：适用单体和AOE场景

**AOE场景使用提示**：
- 插件循环自动维护雷霆一击Debuff（单体+AOE通用）
- 多目标时手动穿插：冲击波（15怒气）、复仇（格挡触发频繁）
- 保持盾牌格挡激活，最大化复仇触发频率

**已知限制**：
- ❌ `target.is_casting`：目标施法检测暂不支持（盾牌反击需手动）→ 见 [TODO.md](../TODO.md#12-目标施法检测-targetis_casting--低优先级)
- ⚠️ `active_enemies`：多目标计数固定为1（AOE场景需手动调整循环）→ 见 [TODO.md](../TODO.md#11-多目标计数-active_enemies--中优先级)

**注意**：嘲讽需手动使用（多坦场景需换T，避免误嘲）

## 战斗策略

### 开局序列

**单体Boss开怪**：
```
1. 冲锋 (Charge) - 进入近战范围
2. 盾牌格挡 (Shield Block) - 立即开启100%格挡
3. 盾牌猛击 (Shield Slam) - 建立初始威胁
4. 雷霆一击 (Thunder Clap) - 挂Debuff
5. 复仇 (Revenge) - 格挡触发后立即使用
6. 毁灭打击 (Devastate) - 开始叠加破甲
```

**AOE小怪群**：
```
1. 冲锋 (Charge) - 进入怪群中心
2. 雷霆一击 (Thunder Clap) - 建立群体威胁+挂Debuff
3. 盾牌格挡 (Shield Block) - 开启格挡（触发复仇关键）
4. 复仇 (Revenge) - 循环打击（格挡后频繁触发，高DPE）
5. 挫志怒吼 (Demoralizing Shout) - 降低所有敌人攻击
6. 冲击波 (Shockwave) - 穿插使用（185%威胁+眩晕控制）
```

**注意**：
- 插件会自动维持雷霆一击Debuff，无需手动卡CD
- AOE时优先使用复仇（5怒气）> 冲击波（15怒气）> 毁灭（15怒气）
- 盾牌格挡在AOE中价值极高（多次格挡=多次复仇触发）

### 血量管理

**血量阈值应对**：
- **100-50%**：正常循环，维持主动防御
- **50-35%**：
  - 准备狂怒回复（怒火专注天赋：额外恢复15%生命+吸收盾）
  - 通知治疗准备
- **35-20%**：
  - **破釜沉舟** - 增加30%最大生命值
  - **狂怒回复** - 配合使用（活力天赋：免疫下次致命伤害）
- **<20%**：
  - **盾墙** - 60%减伤，终极保命
  - **狂怒回复** - 如CD就绪，提供免死保护

**生存技能组合**：
- 极限生存：盾墙 + 盾牌格挡 + 狂怒回复
- 紧急自救：破釜沉舟 + 狂怒回复（+30%血量+治疗+吸收盾+免死）
- 团队危机：集结呐喊（督军天赋，全团+20%血量10秒）
- 法术伤害：法术反射（单体）或移动躲避

### 威胁管理

**单体威胁优先级**：
```
复仇（5怒气，DPE最高）
  ↓
盾牌猛击（20怒气，主要威胁，CD 6秒）
  ↓
毁灭打击（15怒气，填充+破甲，4T1威胁+30%）
  ↓
英勇打击（怒气≥60时，泄怒用）
```

**AOE威胁技巧**：
- **雷霆一击**：插件自动维护Debuff，无需手动管理
- **冲击波**：手动穿插使用，威胁值=伤害×185%（泰坦重铸强化）+3秒眩晕
- **复仇**：配合盾牌格挡高频触发，AOE场景DPE最高技能
- **挫志怒吼**：降低所有敌人攻击强度（-AP/AP×54）
- **英勇打击**：怒气≥60时泄怒，顺劈当前目标

**实战要点**：
- 单体循环已包含雷霆一击，可直接用于AOE
- 多目标时手动增加冲击波频率（威胁+控制）
- 保持盾牌格挡激活（多次格挡=多次复仇）
- 怒气充足时优先复仇>冲击波>毁灭

**OT应对**：
1. 立即嘲讽拉回目标（手动使用）
2. 盾牌猛击 + 英勇打击快速建立威胁
3. 通知DPS降低输出

**捍卫天赋使用**（泰坦重铸改动）：
- 给另一个坦克使用捍卫时，如果目标处于坦克姿态（防御姿态/熊形态/冰霜灵气/正义之怒），不再降低其威胁值
- 双坦配合更灵活，主副坦可自由切换

### 怒气分配策略

**怒气来源**：
- 受到伤害（主要来源）
- 盾牌格挡触发（额外产生）
- 格挡/招架/躲闪（触发复仇机会）

**怒气使用原则**：
- **20-40怒气**：正常循环（盾牌猛击、复仇优先）
- **40-60怒气**：可以使用毁灭打击填充
- **≥60怒气**：**立即英勇打击泄怒**
  - 防止怒气浪费（上限100）
  - 不占GCD，不影响循环节奏
  - 将溢出怒气转化为威胁输出

## 常见问题

**Q: 为什么复仇优先级高于盾牌猛击？**  
A: **DPE（每怒气威胁值）更高**。复仇5怒气必爆高威胁，盾牌猛击20怒气，威胁/怒气比例约3-4倍。格挡/招架/躲闪后立即可用，怒气效率极高。

**Q: 什么时候使用盾墙和破釜沉舟？**  
A: **血量阈值自动触发**。破釜沉舟<35%，盾墙<20%，配合狂怒回复（活力天赋提供免死）。插件会高亮显示，看到立即使用。

**Q: 怒气≥60为什么要用英勇打击？**  
A: **防止怒气浪费，最大化威胁**。防御战怒气上限100，超过会浪费。英勇打击不占GCD不影响循环，将溢出怒气转化为额外威胁。

**Q: 4T1套装值得牺牲属性吗？**  
A: **绝对值得**。毁灭打击威胁+30%，这是主要填充技能（15怒气无CD）。即使单件属性差10-15%，套装效果收益仍然很高。优先凑齐4T1。

**Q: 嘲讽为何不自动触发？**  
A: **多坦场景需要手动控制**。换T场景需要主副坦协调切换，自动嘲讽可能打乱团队策略，手动使用更灵活。

**Q: AOE场景如何调整循环？**  
A: **单体循环已通用**。插件自动维护雷霆一击Debuff，AOE时手动穿插冲击波（185%威胁+眩晕）。关键是保持盾牌格挡激活，最大化复仇触发频率（5怒气必爆，AOE最高DPE）。

**Q: 盾牌反击如何使用？**  
A: **目标施法时手动使用**。插件暂不支持自动检测目标施法状态，需玩家观察Boss施法条手动打断。优先打断：群体伤害技能、治疗法术、关键Buff。

**Q: 狂怒回复有多强？**  
A: **三重增强机制**（怒火专注天赋）：额外恢复15%生命+不消耗激怒+100%过量治疗转化吸收盾（持续15秒）。配合活力天赋在激活时免疫下次致命伤害。
