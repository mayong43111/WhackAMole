# 泰坦重铸 (Titan Forged) 冰霜死亡骑士输出逻辑文档 - T1阶段

## 关于服务器版本 - 泰坦重铸 (Titan Forged)
**确认信息**: 这是一个基于 3.3.5a (巫妖王之怒) 核心的中国区特别版本服务器。
- **技能系统**: 使用完整的巫妖王之怒技能机制（符文系统、符文能量、杀戮机器、冰冷触摸等）
- **装备系统**: 使用魔改的经典旧世T1套装（当前阶段），套装效果针对WOTLK机制重新设计
本插件逻辑完全遵循 3.3.5a 机制，并针对服务器的**魔改T1套装**特效进行优化。

## 核心机制
3.3.5a 版本的冰霜死亡骑士是一个基于**符文管理**和**符文能量优化**的爆发型DPS系统。

### 属性说明 (Attributes)
**关于冰霜死亡骑士的属性**:
*   冰霜死亡骑士在 3.3.5a 是标准的力量近战DPS，依赖力量、护甲穿透和暴击。
*   **符文系统**: 6个符文（2鲜血、2冰霜、2邪恶），每个符文10秒冷却。死亡符文可用于任何类型技能。
*   **符文能量**: 最大130点，通过消耗符文的技能生成（每个符文生成10点符文能量）。

### 属性优先级 (Stats Priority)
1.  **命中 (Hit)**: 8% (达标) - 物理近战命中需要8%
    *   团队配置完善（德莱尼 +1%）时，装备提供 7% 即可
2.  **精准 (Expertise)**: 26 (达标) - 减少被招架和躲闪
3.  **力量 (Strength)**: 核心属性，提供攻击强度
4.  **护甲穿透 (Armor Penetration)**: T1阶段后期开始堆叠，收益随装备提升
5.  **暴击 (Crit)**: 重要属性，触发杀戮机器（Killing Machine）等效果
6.  **急速 (Haste)**: 提升攻击速度和符文恢复，收益良好

### 套装特效 - 魔改T1 (无畏套装)
泰坦重铸服务器的 T1 套装针对WOTLK机制重新设计：

*   **2T1**: 【寒冬号角】在使用时额外产生 **5点符文能量**
    *   **机制说明**: 
        - **寒冬号角 (Horn of Winter)**: 无CD，提升攻击强度和符文能量（基础10点）
        - **2T1后**: 产生15点符文能量（+50%符文能量生成）
    *   **实战价值**: 
        - 符文能量是冰霜DK的核心资源，用于施放冰霜打击（Frost Strike）
        - 额外5点符能意味着可以更频繁地使用冰霜打击
        - **收益**: 约提升3-5%符文能量生成效率，整体DPS提升约1-2%
    *   **优先级**: 中等。主要为符能管理优化和凑4件套的过渡效果
    
*   **4T1**: 【凋零缠绕】和【冰霜打击】造成的伤害提高 **5%**
    *   **机制说明**: 
        - **凋零缠绕 (Obliterate)**: 冰霜DK核心技能，消耗1冰霜+1邪恶符文
        - **冰霜打击 (Frost Strike)**: 符文能量消耗技能（40点），高频率使用
    *   **实战意义**: 
        - 这两个技能占冰霜DK总伤害的50-60%
        - 5%伤害提升直接转化为2.5-3%总体DPS提升
        - **数学模型**: 假设凋零缠绕占30%伤害，冰霜打击占25%伤害
            - 无套装: 100% DPS
            - 4T1后: (30% × 1.05) + (25% × 1.05) + 45% = 103.25% DPS (+3.25%)
    *   **优化策略**: 4T1后优先保证凋零缠绕和冰霜打击的使用频率
    
*   **重点**: 魔改T1专注于强化**核心输出技能**，2T1增强资源生成，4T1直接提升主要伤害技能

## 逻辑分析 (优先级队列)

### 常规单体循环

1.  **寒冬号角 (Horn of Winter)**
    *   **条件**: 符文能量<60（避免浪费）
    *   **理由**: 产生符文能量的主要手段。**2T1后产生15符能**，提升符能生成效率
    *   **重要性**: 高频率使用，保持符能充足

2.  **冰冷触摸 (Icy Touch)**
    *   **条件**: 冰霜疫病不存在或即将结束（<3秒）
    *   **理由**: 施加冰霜疫病，重要DoT之一
    *   **优先级**: 保持疫病覆盖

3.  **疫病打击 (Plague Strike)**
    *   **条件**: 血腥疫病不存在或即将结束（<3秒）
    *   **理由**: 施加血腥疫病，重要DoT之二
    *   **优先级**: 保持疫病覆盖

4.  **凋零缠绕 (Obliterate) - 杀戮机器触发**
    *   **条件**: 杀戮机器Buff存在
    *   **理由**: 保证凋零缠绕暴击，最大化伤害。**4T1后+5%伤害**
    *   **优先级**: 杀戮机器触发时最高优先级

5.  **冰霜打击 (Frost Strike) - 杀戮机器触发**
    *   **条件**: 杀戮机器Buff存在，符能>=40
    *   **理由**: 保证冰霜打击暴击。**4T1后+5%伤害**
    *   **优先级**: 杀戮机器触发时高优先级（次于凋零缠绕）

6.  **凋零缠绕 (Obliterate) - 常规**
    *   **条件**: 有1冰霜+1邪恶符文（或死亡符文）
    *   **理由**: 核心高伤害技能。**4T1后+5%伤害**
    *   **优先级**: 主要符文消耗技能

7.  **冰霜打击 (Frost Strike) - 常规**
    *   **条件**: 符能>=40
    *   **理由**: 主要符能消耗技能。**4T1后+5%伤害**
    *   **优先级**: 符能充足时使用

8.  **凛风冲击 (Howling Blast)**
    *   **条件**: 有冰霜符文或死亡符文
    *   **理由**: AOE伤害，触发冰冷触摸Buff
    *   **优先级**: 单目标时作为填充技能

9.  **寒冬号角 (Horn of Winter) - 填充**
    *   **条件**: 符文能量<100
    *   **理由**: 无CD，随时可用。**2T1后15符能**
    *   **优先级**: 符文CD时的填充技能

### AOE循环

1.  **凛风冲击 (Howling Blast)**: 主要AOE输出
2.  **瘟疫打击 (Pestilence)**: 传播疫病到多个目标
3.  **冰冷触摸 (Icy Touch)**: 施加冰霜疫病
4.  **疫病打击 (Plague Strike)**: 施加血腥疫病
5.  **凋零缠绕 (Obliterate)**: 高伤害单体技能（主目标）
6.  **冰霜打击 (Frost Strike)**: 符能消耗

**4T1优势**: 凋零缠绕和冰霜打击+5%伤害，单体和AOE都有提升

## 常见问题 (FAQ)

### 为什么寒冬号角优先级这么高？
寒冬号角无CD，是冰霜DK主要的符能生成手段（除了消耗符文）。2T1后产生15符能（+50%），高频使用可以显著提升符能生成效率，保证冰霜打击的高频率使用。符能是冰霜DK的核心资源，充足的符能才能最大化DPS。

### 4T1的凋零缠绕和冰霜打击+5%有多重要？
**非常重要**！这两个技能占冰霜DK总伤害的50-60%：
- 凋零缠绕：核心符文消耗技能，占总伤害约30%
- 冰霜打击：核心符能消耗技能，占总伤害约25%
- 5%伤害提升意味着整体DPS提升约3.25%，是非常可观的提升

### 杀戮机器触发时应该用凋零缠绕还是冰霜打击？
**优先凋零缠绕**！因为：
1. 凋零缠绕单次伤害更高（消耗2个符文）
2. 凋零缠绕占总伤害比例更大（约30% vs 25%）
3. 杀戮机器保证暴击，凋零缠绕暴击的收益更高
4. 冰霜打击可以用符能随时施放，而凋零缠绕需要等待符文CD

### 疫病为什么要维持？
冰霜疫病+血腥疫病共占总伤害10-15%，虽然不如凋零缠绕和冰霜打击高，但仍是重要的伤害来源。更重要的是，疫病的存在会影响其他技能的效果（如瘟疫打击的传播）。因此必须保持高覆盖率。

### T1阶段冰霜死亡骑士的定位如何？
T1阶段冰霜DK是顶级近战物理DPS，爆发能力强（杀戮机器机制），持续输出稳定。魔改T1的4件套直接强化核心技能，使其在单体和多目标场景都表现优秀。同时提供【冰冷触摸】光环（20%近战攻击速度），是优秀的近战DPS职业。
