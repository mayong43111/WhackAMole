# 泰坦重铸 (Titan Forged) 武器战输出逻辑文档 - T1阶段

## 关于服务器版本 - 泰坦重铸 (Titan Forged)
**确认信息**: 这是一个基于 3.3.5a (巫妖王之怒) 核心的中国区特别版本服务器。
- **技能系统**: 使用完整的巫妖王之怒技能机制（血涌、猝死、撕裂、压制等）
- **装备系统**: 使用魔改的经典旧世T1套装（当前阶段），套装效果针对WOTLK机制重新设计
本插件逻辑完全遵循 3.3.5a 机制，并针对服务器的**魔改T1套装**特效进行优化。

## 核心机制
3.3.5a 版本的武器战并不是一个死板的循环，而是一个基于**优先级**的动态系统。

### 属性说明 (Attributes)
**关于护甲穿透 (ArP)**:
*   很多玩家在面板上直接看不到“护甲穿透”，这是 3.3.5a 客户端 UI 的通病。
*   **查看方法**: 打开人物属性 (C) -> 在详细数据的下拉菜单中选择 **近战 (Melee)** -> 鼠标移动到 **命中** 或 **暴击** 一栏上，或者查找单独的“护甲穿透等级”。
*   **硬指标**: 1400 穿透等级 = 100% 无视护甲 (俗称“满破”)。这是本服后期依然生效的核心机制。

### 属性优先级 (Stats Priority)
**特别说明**: 在游戏前期 (例如纳克萨玛斯/奥杜尔阶段)，你可能很难找到带护甲穿透的装备。这是正常的。
*   **无破甲装备时**: 优先堆 **力量 (Strength)** 和 **暴击 (Crit)**。
*   **后期目标**: 当你能获取 TOC/ICC 级别装备 (物品等级 230+) 时，护甲穿透才会大量出现。

1.  **命中 (Hit)**: 8% (达标)
2.  **精准 (Exp)**: 26 (达标)
3.  **护甲穿透 (ArP)**:
    *   **前期**: 随缘，有就穿，没有不强求。
    *   **后期**: 必须硬满破 1400 (优先级 > 力量)。
4.  **暴击 (Crit)**: 前期核心属性。
5.  **力量 (Strength)**: 基础收益最高，前期核心。

### 检测命令
如果你不确定自己的破甲属性，可以在游戏聊天框输入以下命令查询：
`/run print("当前破甲等级:", GetCombatRating(25))`

### 套装特效 - 魔改T1 (无畏套装)
泰坦重铸服务器的 T1 套装针对WOTLK机制重新设计：
*   **2T1**: 【猛击】和【旋风斩】的技能伤害提高 **3%**
    *   **实战价值**: 直接增强两个常用填充/AOE技能的伤害。猛击是武器战最主要的填充技能，3%提升是稳定收益
    *   **优先级**: 中等。主要为凑4件套的过渡效果
*   **4T1**: 激活【血涌】的几率提高 **5%**，激活【猝死】的几率提高 **2%**
    *   **机制说明**: 
        - **血涌 (Taste for Blood)**: 撕裂每6秒跳一次时有几率触发，让你可以使用压制。基础几率约30%，+5%后达到35%
        - **猝死 (Sudden Death)**: 自动攻击时有几率触发，让你可以无视血量使用斩杀。基础几率约6-9%，+2%后达到8-11%
    *   **实战意义**: 
        - **更多压制**: 5怒气必爆的高效技能，触发率提升约16.7%（30%→35%）
        - **更多猝死斩杀**: 20%以上血量阶段也能打出高伤害斩杀，DPS显著提升
    *   **数学模型**: 假设战斗持续60秒
        - 撕裂跳动10次，血涌触发次数: 10 × 30% = 3次 → 10 × 35% = 3.5次 (+16.7%)
        - 自动攻击约40次，猝死触发次数: 40 × 7% = 2.8次 → 40 × 9% = 3.6次 (+28.6%)
    *   **连锁反应**: 更多压制/猝死 → 更多怒气转化 → 更高DPS → Boss战优势明显
*   **重点**: 与原版T10不同，魔改T1专注于强化**核心Proc机制**（血涌/猝死）而非单纯的伤害百分比

*   **猝死 (Sudden Death)** (Buff): ID `52437`
    - 自动攻击时有**6-9%基础几率**触发（4T1后提升至8-11%）
    - 允许无视血量使用斩杀，并保留10点怒气

*   **撕裂 (Rend)** (Debuff): ID `47465`。必须全程维持，是触发血涌的唯一途径。

## 逻辑分析 (优先级队列)

### 阶段一：常规阶段 (> 20% 血量)

1.  **撕裂 (Rend)**
    *   **条件**: 目标身上没有撕裂，或剩余时间 < 3秒。
    *   **理由**: 触发血涌的唯一途径。**4T1后触发率提升至35%**，每6秒有更高几率获得压制机会。
    *   **重要性**: 即使在满破甲环境下，撕裂依然是武器战的核心技能，绝不能断。

2.  **压制 (Overpower)**
    *   **条件**: `血涌` Buff 存在，或技能亮起。
    *   **理由**: 5怒气必爆，无法被格挡/招架。**4T1后触发次数增加约16.7%**，是武器战最高效的DPE技能。
    *   **细节**: 如果 Buff 剩余时间 < 1.5秒，必须提到最高优先级以免浪费。
    *   **4T1优势**: 更频繁的触发意味着更稳定的怒气循环和更高的整体DPS。

3.  **斩杀 (Execute) - 猝死触发**
    *   **条件**: `猝死` Buff 存在。
    *   **理由**: 保留怒气的一发大伤害。**4T1后触发率提升至8-11%**，意味着每场战斗多打出约25-30%的猝死斩杀。
    *   **战略意义**: 将高伤害斩杀技能提前到20%以上血量使用，大幅提升常规阶段DPS。

4.  **致死打击 (Mortal Strike)**
    *   **条件**: 冷却就绪。
    *   **理由**: 核心伤害来源，挂致死 Debuff，刷新重伤。

5.  **利刃风暴 (Bladestorm)**
    *   **条件**: 冷却就绪，且**不是**猝死/压制触发期。
    *   **时机**: 尽量在该技能CD好时使用。但如果手里有压制/猝死，先打出去再转，否则转完了 Buff 也没了。

6.  **猛击 (Slam)**
    *   **条件**: 站桩（未移动），且怒气 > 20。
    *   **理由**: 填充技能。**2T1后伤害提升3%**，使其成为更有价值的怒气转化选择。
    *   **注意**: 需要读条，高急速或高延迟环境下需谨慎使用。

### 阶段二：斩杀阶段 (< 20% 血量)

**错误观念**: 到了 20% 就无脑按斩杀。
**正确思路**: 在 3.3.5 版本，满怒斩杀 DPE 很高，但低怒斩杀甚至不如致死。你需要维持怒气循环。

1.  **撕裂 (Rend)**: 依然保持！为了给压制提供触发。
2.  **压制 (Overpower)**: 依然打！5怒气打出必爆伤害，性价比远超低怒斩杀，是攒怒气打满怒斩杀的关键。
3.  **致死打击 (Mortal Strike)**: 如果怒气 > 40，可以用致死替代一发低怒斩杀（为了保持重伤）。或者仅在技能CD时作为填充。
    *   *Luminary 策略*: 采用 **压制 > 斩杀** 的策略。
4.  **斩杀 (Execute)**: 泄怒主要手段。

## 拟定 Lua 代码逻辑 - T1阶段优化版

```lua
-- 实际APL逻辑（已实装）

-- 1. 维持撕裂 (最高优先级 - 触发血涌的唯一途径)
-- 条件：剩余时间<3秒 且 目标存活>6秒
if debuff(S_Rend).remains < 3 and target.time_to_die > 6 then
    return 4 -- 撕裂
end

-- 2. 压制 (第二优先级 - 5怒气必爆，4T1后35%触发率)
-- 条件：技能可用（血涌Buff触发或目标躲闪/招架）
if spell(S_Overpower).ready then
    return 2 -- 压制
end

-- 3. 猝死斩杀 (20%以上血量 - 4T1后8-11%触发率)
-- 条件：猝死Buff存在 且 目标血量≥20%
if buff(B_SuddenDeath).up and target.health_pct >= 20 then
    return 3 -- 斩杀
end

-- 4. 常规斩杀 (20%以下血量 - 需要足够怒气)
-- 条件：目标血量<20% 且 怒气≥30（避免低效斩杀）
if target.health_pct < 20 and player.power.rage.current >= 30 then
    return 3 -- 斩杀
end

-- 5. 致死打击 (核心伤害技能)
-- 条件：CD就绪
if spell(S_MortalStrike).ready then
    return 1 -- 致死打击
end

-- 6. 利刃风暴 (爆发技能 - 智能使用)
-- 条件：CD就绪 且 没有猝死/压制Buff（避免浪费Proc）
if spell(S_Bladestorm).ready and not buff(B_SuddenDeath).up and not spell(S_Overpower).ready then
    return 6 -- 利刃风暴
end

-- 7. 猛击 (填充技能 - 2T1后+3%伤害)
-- 条件：站桩 且 怒气≥20 且 目标血量≥20%（斩杀阶段不用）
if not player.moving and player.power.rage.current >= 20 and target.health_pct >= 20 then
    return 5 -- 猛击
end

-- 8. 英勇打击 (怒气溢出时泄怒)
-- 条件：怒气≥60 且 目标血量≥20%（不占GCD，替代下次平砍）
if player.power.rage.current >= 60 and target.health_pct >= 20 then
    return 7 -- 英勇打击
end

-- 9. 雷霆一击 (多目标场景)
-- 条件：目标缺少Debuff 且 存在多个敌人
if debuff(D_ThunderClap).down and active_enemies >= 2 then
    return 8 -- 雷霆一击
end
```

### 代码逻辑说明

**优先级设计理念**：
1. **撕裂永不断** - 血涌触发的基础，4T1使其价值更高
2. **压制立即打** - DPE最高，5怒气必爆，4T1后触发率35%
3. **智能斩杀** - 区分猝死斩杀（20%+）和常规斩杀（20%-），避免低怒气浪费
4. **怒气管理** - 猛击≥20怒气，英勇打击≥60怒气，确保高效转化

**4T1套装优化**：
- 压制优先级极高（第2位），充分利用35%触发率
- 猝死斩杀独立判断，延长高DPS输出窗口
- 猛击享受2T1的+3%伤害加成

**关键改进**：
- 移除雷霆一击的高优先级（原来错误地放在压制之前）
- 分离猝死斩杀逻辑，避免20%以上血量浪费怒气在低效斩杀上
- 添加英勇打击作为怒气溢出的泄怒手段
- 利刃风暴添加智能检测，避免在有Proc时使用而浪费

## 泰坦重铸特色说明 - T1阶段
*   **4T1核心价值**: 务必获取4件套T1！+5%血涌和+2%猝死触发率看似不多，但实际提升约20-30%的Proc总量。
*   **血涌提升**: 35%触发率意味着撕裂每跳3次就能触发1次压制，而非原来的每3.3次。更稳定的压制循环 = 更高的怒气效率。
*   **猝死质变**: +2%触发率使得常规阶段也能频繁打出斩杀，大幅缩短战斗时间。在Boss战中尤为明显。
*   **2T1价值**: 猛击+3%伤害虽然不如4件套重要，但作为主要填充技能，这是稳定的DPS提升。
*   **撕裂重要性**: T1阶段撕裂不仅是触发血涌的唯一途径，其自身的流血伤害在前期装备下占比也更高。绝对不能断。
*   **装备优先级**: 命中8% > 精准26 > 暴击/力量 > 护甲穿透（T1阶段破甲装备稀少，不强求）
*   **套装优先**: 即使单件属性不如散件，也要优先凑齐4T1。Proc提升带来的收益远超属性差异。
