# 泰坦重铸 (Titan Forged) 冰法输出逻辑文档 - T1阶段

## 关于服务器版本 - 泰坦重铸 (Titan Forged)
**确认信息**: 这是一个基于 3.3.5a (巫妖王之怒) 核心的中国区特别版本服务器。
- **技能系统**: 使用完整的巫妖王之怒技能（寒冰箭、寒冰宝珠、深度冻结、冰枪术等）
- **装备系统**: 使用魔改的经典旧世T1套装（当前阶段），套装效果针对WOTLK机制重新设计
- **冰法特色**: 泰坦服引入了"**寒冰宝珠**"机制，打造全新"战斗法师"玩法

本插件逻辑遵循 3.3.5a 冰法机制，并针对服务器的**魔改T1套装**和**寒冰宝珠**特效进行优化。

## 核心机制
冰法在泰坦服经历了重大改版，从传统的站桩读条流派变成了灵活的"战斗法师"。

### 泰坦服冰法特色改动（2026年1月）

#### 1. 寒冰宝珠（Frostfire Orb）- 核心机制
- **来源**: 原"冰枪术雕文"更名为"寒冰宝珠雕文"
- **效果**: 增加寒冰宝珠的爆炸伤害 10%
- **机制**: 
  * 投掷寒冰宝珠，对路径上的敌人造成持续伤害
  * 宝珠爆炸时造成大量AOE伤害
  * 冷却时间短，可频繁使用
  * **近战范围**内伤害最高，鼓励接近目标战斗

#### 2. 天赋调整
**冷锋（Cold Front）**:
- 强化冰霜新星伤害：20/40/60/80/100%（原为15/30/45/60/75%）
- 强化冰锥术伤害：类似幅度提升

**寒冰迅捷（Ice Floes）**:
- 减少寒冰箭施法时间：0.15/0.3秒（原为0.1/0.2秒）

**彻骨之寒（Piercing Ice）**:
- 提升寒冰箭/霜火球/冰枪术伤害：2%/4%/6%/8%/10%（原为1%/2%/3%/4%/5%）

**脑部冻结（Brain Freeze）**:
- 触发后使霜火球瞬发并强化

**召唤水元素**:
- **重要变更**: 水元素不再受益于小队增益（Buff）
- 这意味着在团队环境中，水元素的DPS贡献降低

### 属性说明 (Attributes)
**关于面板**:
*   **命中 (Hit)**: 17% (达标)。
    *   在团队配置完善（暗牧/鸟德 +3%， 德莱尼 +1%）的情况下，装备提供 13%-14% 即可
    *   **2T1加成**: 有2件套后额外获得1%命中，装备只需12%-13%即可达标
*   **暴击 (Crit)**: 冰法的重要属性。
    *   **机制**: 暴击触发**深度冻结（Fingers of Frost）**，使下次冰枪术/深度冻结必定暴击
    *   **机制2**: 连续两次寒冰箭/霜火球暴击触发**脑部冻结（Brain Freeze）**，下次霜火球瞬发
    *   **目标**: 泰坦T1阶段，暴击30-40%是合理目标
*   **急速 (Haste)**: 冰法的核心属性。
    *   由于寒冰箭读条时间长，急速收益极高
    *   寒冰宝珠流派下，急速也影响宝珠投掷频率
    *   **目标**: 在保证命中达标后，优先堆急速
*   **法术强度 (Spell Power)**: 基础伤害来源。

### 套装特效 - 魔改T1 (奥术师套装)
泰坦重铸服务器的 T1 套装针对WOTLK机制重新设计：
*   **2T1**: 提高**法术命中率1%**，并降低**法术消耗1%**
    *   **实战价值**: 命中接近达标后可洗出装备命中换成急速/暴击
    *   **优先级**: 中等。主要为凑4件套的过渡效果
*   **4T1**: **【寒冰宝珠】爆炸伤害提高15%**（推测）
    *   **注意**: 由于是服务器魔改内容，具体数值可能需要实测确认
    *   **机制说明**: 强化寒冰宝珠的爆炸AOE伤害
    *   **实战意义**: 配合寒冰宝珠雕文（+10%），总共获得约25%的宝珠伤害提升
*   **重点**: 魔改T1套装专注于强化**寒冰宝珠机制**，打造近战法师玩法

## 逻辑分析 (优先级队列)

### 寒冰宝珠流派（战斗法师）- 推荐

这是泰坦服特色玩法，利用寒冰宝珠的短CD和高伤害打造灵活战斗风格。

1.  **镜像 (Mirror Image)**
    *   **条件**: 冷却就绪
    *   **理由**: 威胁转移+额外DPS

2.  **寒冰宝珠 (Frostfire Orb)**
    *   **条件**: 冷却就绪
    *   **理由**: 核心输出技能，CD好了立刻使用
    *   **注意**: 尽量在近战距离使用以获得最高伤害

3.  **深度冻结 (Deep Freeze)**
    *   **条件**: 冷却就绪 且 (目标冻结 或 有"深度冻结"Buff)
    *   **理由**: 高伤害爆发技能

4.  **冰枪术 (Ice Lance) - 深度冻结触发**
    *   **条件**: 有"深度冻结"Buff
    *   **理由**: 瞬发高伤，必定暴击

5.  **霜火球 (Frostfire Bolt) - 脑部冻结触发**
    *   **条件**: 有"脑部冻结"Buff
    *   **理由**: 瞬发高伤，比寒冰箭更强

6.  **冰锥术 (Cone of Cold)**
    *   **条件**: 在近战范围内
    *   **理由**: 配合寒冰宝珠流派，近战输出

7.  **寒冰箭 (Frostbolt)**
    *   **条件**: 站桩填充
    *   **理由**: 主要输出技能，读条期间等待触发

8.  **冰枪术 (Ice Lance) - 移动填充**
    *   **条件**: 移动中
    *   **理由**: 瞬发技能，移动输出

### 传统冰法流派（可选）

如果不适应寒冰宝珠流派，可以使用传统打法：

1.  **镜像 (Mirror Image)** - CD好就用
2.  **深度冻结 (Deep Freeze)** - 有Buff就用
3.  **冰枪术 (Ice Lance)** - 有"深度冻结"Buff
4.  **霜火球 (Frostfire Bolt)** - 有"脑部冻结"Buff
5.  **寒冰箭 (Frostbolt)** - 主要填充
6.  **冰枪术 (Ice Lance)** - 移动填充

## 拟定 Lua 代码逻辑 - T1阶段（寒冰宝珠流派）

```lua
-- 1. 爆发：镜像
if spell(S_MirrorImage).ready then
    return 8 -- 镜像
end

-- 2. 核心输出：寒冰宝珠
if spell(S_FrostfireOrb).ready then
    return 2 -- 寒冰宝珠
end

-- 3. 深度冻结爆发
if spell(S_DeepFreeze).ready and buff(B_FingersOfFrost).up then
    return 3 -- 深度冻结
end

-- 4. 消耗深度冻结Buff
if buff(B_FingersOfFrost).up then
    return 4 -- 冰枪术（深度冻结）
end

-- 5. 消耗脑部冻结Buff
if buff(B_BrainFreeze).up then
    return 5 -- 霜火球（脑部冻结）
end

-- 6. 近战AOE（战斗法师玩法）
if target.distance < 10 and spell(S_ConeOfCold).ready then
    return 6 -- 冰锥术
end

-- 7. 移动填充
if player.moving then
    return 4 -- 冰枪术
end

-- 8. 主要填充：寒冰箭
return 1 -- 寒冰箭
```

**核心思路**: 
- **寒冰宝珠优先**: CD好了立即使用，这是泰坦服特色
- **消耗Buff**: 优先消耗"深度冻结"和"脑部冻结"触发
- **近战输出**: 在安全的情况下接近目标，使用冰锥术配合宝珠
- **灵活移动**: 利用瞬发技能保持移动输出
- **主要输出**: 寒冰箭spam等待触发

## 泰坦重铸特色说明 - T1阶段

### 寒冰宝珠流派优势
*   **高爆发**: 寒冰宝珠CD短（约15秒），爆炸伤害高
*   **机动性**: 大量瞬发技能，可以边跑边打
*   **近战法师**: 鼓励接近目标战斗，与传统法师玩法不同
*   **AOE优势**: 宝珠+冰锥术在多目标战斗中表现出色

### 4T1核心价值
*   务必获取4件套T1！寒冰宝珠伤害提升15%（推测）配合雕文+10%，总共25%提升
*   这是泰坦服冰法的核心机制，没有4T1套装效果会大打折扣

### 装备优先级
*   **命中17%（达标）** > **急速（极限堆）** > **暴击** > **法伤**
*   急速对寒冰箭和宝珠CD都有收益，是冰法的最重要属性
*   利用2T1的+1%命中，装备只需12-13%命中即可

### 天赋要点
*   必点：**寒冰迅捷**（减少寒冰箭施法时间）
*   必点：**彻骨之寒**（提升冰霜伤害10%）
*   必点：**冷锋**（强化冰锥术和冰霜新星）
*   必点：**深度冻结**和**脑部冻结**触发天赋

### 雕文选择
*   **大雕文**:
    - 寒冰宝珠雕文（+10%宝珠伤害，必带）
    - 寒冰箭雕文（减少施法时间或增加伤害）
    - 深度冻结雕文（缩短CD或增加伤害）
*   **小雕文**: 质量较低，根据个人喜好选择

## 战斗技巧

### 开局起手
1.  **预读寒冰箭** → 进入战斗
2.  **立即施放寒冰宝珠** → 爆发输出
3.  **镜像** → 威胁转移
4.  **继续寒冰箭** → 等待触发

### 爆发期
1.  寒冰宝珠CD好了立刻用
2.  消耗所有"深度冻结"和"脑部冻结"Buff
3.  接近目标使用冰锥术（如果安全）
4.  深度冻结+冰枪术combo

### 移动战
1.  优先消耗Buff（霜火球/冰枪术都是瞬发）
2.  使用冰枪术填充
3.  利用冰霜新星创造距离
4.  寒冰宝珠可以边跑边丢

### AOE战斗
1.  多个目标聚集时，寒冰宝珠伤害最大化
2.  冰霜新星冻结 → 冰锥术爆发
3.  暴风雪（长时间AOE）
4.  冰枪术收尾低血目标

## 常见问题 (FAQ)

### 寒冰宝珠流派和传统冰法有什么区别？
**传统冰法**: 站桩读寒冰箭，偶尔用冰枪术，重度依赖"深度冻结"触发
**寒冰宝珠流派**: 高机动性，频繁使用宝珠和近战技能，更像"战斗法师"

泰坦服的改动鼓励冰法接近目标战斗，打破了传统远程法师的固定模式。

### 为什么要接近目标？
寒冰宝珠在近距离爆炸时伤害最高，配合冰锥术可以打出更高DPS。但要注意Boss的近战AOE技能，合理控制距离。

### 水元素还值得召唤吗？
虽然水元素不再受益于团队Buff，但仍然提供额外DPS和冰冻触发。在单人任务中仍然有用，团队本中收益降低。

### T1套装的4件套效果是多少？
**推测是+15%寒冰宝珠伤害**，但具体数值需要实测确认。泰坦服的魔改数值可能与文档不同。

### 急速堆多少合适？
理论上越高越好。寒冰箭基础2.5秒读条，堆到1.5秒是理想目标。同时急速也影响宝珠CD和GCD。

### 这个流派PVP适用吗？
**非常适合PVP**！高机动性+控制技能+爆发伤害，冰法在PVP中本就强势，寒冰宝珠流派更是如虎添翼。
