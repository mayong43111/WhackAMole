# 巫妖王之怒 / 泰坦时光(Titan-Forged) 火法输出逻辑文档

## 关于服务器版本 - 泰坦时光 (Titan Time)
**确认信息**: 这是一个基于 3.3.5a (巫妖王之怒) 核心的改版服务器。
本插件逻辑主要遵循 3.3.5a 折磨火 (TTW Fire) 机制，并针对服务器的 T10 套装特效进行了优化。

## 核心机制
火法在 3.3.5a 是后期的王者专精，极其依赖装备属性（暴击）。

### 属性说明 (Attributes)
**关于面板**:
*   **命中 (Hit)**: 17% (达标)。
    *   在团队配置完善（暗牧/鸟德 +3%， 德莱尼 +1%）的情况下，装备提供 13%-14% 即可。
*   **暴击 (Crit)**: 火法的核心属性。
    *   **机制**: 暴击不仅造成 200%+ 伤害（含天赋点燃），还会触发 **法术连击 (Hot Streak)** 赠送瞬发炎爆术。
    *   **目标**: 在泰坦时光服的高装等支持下，无脑堆暴击是常态。
*   **急速 (Haste)**: 软上限即可。
    *   由于 2T10 特效提供 12% 急速，且此时光服可能存在其他加速手段，急速收益在 GCD 溢出（1秒读条）后会锐减。
*   **法术强度 (Spell Power)**: 基础伤害来源。

### 套装特效 (可能有改动)
根据代码注释，泰坦时光服的 T10 套装可能有以下**魔改**：
*   **2T10**: 触发 **法术连击**、**镜像**、**飞弹速射** 时，获得 12% 急速，持续 5秒。 (ID `70753` - 冲破极限)
*   **4T10**: **镜像** 技能现在会触发 **四核强能**，使**法术伤害提高 18%**，持续 30秒。 (ID `70747`)
    *   **重点**: 如果此特效存在，**镜像 (Mirror Image)** 不再只是一个仇恨/微量伤害技能，而是**最强爆发技能**，必须配合嗜血/饰品使用。

## 逻辑分析 (优先级队列)

### 常规单体循环 (Single Target)

1.  **活动炸弹 (Living Bomb)**
    *   **条件**: 目标身上没有活动炸弹，且目标存活时间 > 12秒。
    *   **细节**: 炸弹爆炸是核心伤害来源，且能触发法术连击。**严禁** 在炸弹爆炸前重新施放（会顶掉爆炸伤害）。

2.  **炎爆术 (Pyroblast) - 法术连击**
    *   **条件**: `法术连击` Buff 存在。
    *   **理由**: 免费、瞬发、高伤。是火法输出的核心构件。
    *   **连号处理**: 如果你在读火球时触发了连击，打完火球立刻接炎爆（火球+炎爆几乎同时命中）。

3.  **燃烧 (Combustion)**
    *   **条件**: 冷却就绪，且目标身上有活动炸弹。
    *   **时机**: 尽量配合 4T10 爆发期。但 3.3.5 的燃烧机制是“增加暴击几率直到暴击 3 次”，是一个不稳定的增益，CD 好了就用由于 CD 较短。

4.  **镜像 (Mirror Image) - 4T10 爆发**
    *   **条件**: 冷却就绪，且是 Boss 战/高血量目标。
    *   **理由**: 获得 18% 法伤加成，极强爆发。

5.  **灼烧 (Scorch) - 补易伤**
    *   **条件**: 目标缺少 5% 暴击 Buff (强化灼烧/暗影烈焰)。
    *   **理由**: 5% 暴击对火法至关重要。

6.  **火球术 (Fireball)**
    *   **条件**: 站桩填充。
    *   **理由**: 平稳输出，不仅造成伤害还能叠点燃。

7.  **火冲击 (Fire Blast)**
    *   **条件**: 移动中无法读条。

### AOE 逻辑

1.  **活动炸弹**: 给多目标挂炸弹（利用爆炸伤害）。
2.  **烈焰风暴 (Flamestrike)**: 针对聚集目标 (等级9 + 等级8 轮流使用以叠加地面的 DoT)。
3.  **冲击波 (Blast Wave) / 龙息术 (Dragon's Breath)**: 填充。
4.  **暴风雪 (Blizzard)**: 目标数量极大时。

## 拟定 Lua 代码逻辑

```lua
-- 1. 爆发：镜像 (如果有 4T10 效果)
if set_bonus.tier10_4pc and spell(S_MirrorImage).ready then
    return 6 -- 镜像爆发
end

-- 2. 燃烧
if spell(S_Combustion).ready then
    return 5 -- 燃烧
end

-- 3. 瞬发炎爆
if buff(B_HotStreak).up then
    return 2 -- 炎爆术
end

-- 4. 补炸弹 (确保上一个炸弹已经炸了)
-- 3.3.5a 必须等炸弹结束，不能提前补
if not debuff(S_LivingBomb).up and target.time_to_die > 12 then
    return 1 -- 活动炸弹
end

-- 5. 补灼烧 (如果缺少暴击易伤)
if debuff(D_ImprovedScorch).down and debuff(D_ShadowMastery).down then
    return 4 -- 灼烧
end

-- 6. 移动填充
if player.moving then
    if spell(S_FireBlast).ready then return 7 end -- 火冲
    if spell(S_DragonsBreath).ready then return 8 end -- 龙息
end

-- 7. 火球术 (主要填充)
return 3 -- 火球术
```

## 泰坦时光服特色说明
*   **高暴击收益**: 随着装等提升，火法的暴击率会越来越高，法术连击的触发频率极高。你的循环可能会变成 `炸弹 -> 火球 -> 炎爆 -> 火球 -> 炎爆` 的机关枪模式。
*   **4T10 质变**: 务必获取 4件套 T10。18% 的法伤加成让原本平庸的镜像变成了神技。
*   **饰品利用**: 泰坦服的强力饰品（如 H异物、H鳞片）触发时，尽量覆盖爆发技能。

## 常见问题 (FAQ)

### 为什么不使用霜火箭 (Frostfire Bolt)？
在 3.3.5a 版本中，火法主要有两个流派：**折磨火 (TTW Fire)** 和 **霜火法 (FFB Fire)**。
但是在泰坦时光服（及所有后期高端团队环境）中，**火球术 (Fireball)** 是绝对首选，原因如下：

1.  **伤害成长性 (Scaling)**: 火球术拥有更高的 **法术强度 (SP)** 加成系数。随着泰坦服装备等级的提升，火球术的单发伤害上限远高于霜火箭。
2.  **折磨弱小 (Torment the Weak)**: 火球术天赋（折磨火 20/51/0）可以点出奥术系的 `折磨弱小`，对被减速目标造成 **12% 额外伤害**。在团队本中 BOSS 身上必定有减速效果（DK/战士提供），这 12% 独立增伤是霜火箭无法比拟的。
3.  **命中溢出**: 霜火箭流派的一个主要优势是天赋提供命中。但在泰坦服后期装备下，命中往往已经达标甚至溢出，霜火箭的这个优势不再存在。

**结论**: 除非你处于刚满级且命中严重不足的极端前期，否则请使用 **火球术**。本插件逻辑仅支持火球术流派。
