# 泰坦重铸 (Titan Forged) 火法输出逻辑文档 - T1阶段

## 关于服务器版本 - 泰坦重铸 (Titan Forged)
**确认信息**: 这是一个基于 3.3.5a (巫妖王之怒) 核心的中国区特别版本服务器。
- **技能系统**: 使用完整的巫妖王之怒技能（法术连击、活动炸弹、燃烧、镜像等）
- **装备系统**: 使用魔改的经典旧世T1套装（当前阶段），套装效果针对WOTLK机制重新设计
本插件逻辑遵循 3.3.5a 折磨火 (TTW Fire) 机制，并针对服务器的**魔改T1套装**特效进行优化。

## 核心机制
火法在 3.3.5a 是后期的王者专精，极其依赖装备属性（暴击）。

### 属性说明 (Attributes)
**关于面板**:
*   **命中 (Hit)**: 17% (达标)。
    *   在团队配置完善（暗牧/鸟德 +3%， 德莱尼 +1%）的情况下，装备提供 13%-14% 即可
    *   **2T1加成**: 有2件套后额外获得1%命中，装备只需12%-13%即可达标，省出的命中可洗成暴击
*   **暴击 (Crit)**: 火法的核心属性。
    *   **机制**: 暴击不仅造成 200%+ 伤害（含天赋点燃），还会触发 **法术连击 (Hot Streak)** 赠送瞬发炎爆术。
    *   **目标**: 在泰坦时光服的高装等支持下，无脑堆暴击是常态。
*   **急速 (Haste)**: 软上限即可。
    *   由于 2T10 特效提供 12% 急速，且此时光服可能存在其他加速手段，急速收益在 GCD 溢出（1秒读条）后会锐减。
*   **法术强度 (Spell Power)**: 基础伤害来源。

### 套装特效 - 魔改T1 (奥术师套装)
泰坦重铸服务器的 T1 套装针对WOTLK机制重新设计：
*   **2T1**: 提高**法术命中率1%**，并降低**法术消耗1%**
    *   **实战价值**: 命中接近达标后可洗出装备命中换成暴击；法力消耗降低在长时间战斗中有一定价值
    *   **优先级**: 中等。主要为凑4件套的过渡效果
*   **4T1**: **【专注魔法】效果获得的暴击率提高2%**
    *   **机制说明**: "专注魔法"指触发**法术连击(Hot Streak)**的暴击判定，+2%暴击率意味着更容易触发连击
    *   **实战意义**: 显著提高法术连击的触发频率！
    *   **数学模型**: 假设基础40%暴击率，触发连击需要连续2次暴击
        - 无套装: 0.40 × 0.40 = 16% 连击触发率
        - 有4T1: 0.42 × 0.42 = 17.64% 连击触发率 (**提升约10%触发频率**)
    *   **连锁反应**: 更多瞬发炎爆术 → 更高伤害 → 更多点燃层数 → DPS质变
*   **重点**: 与原版T10不同，魔改T1专注于强化**法术连击机制**而非镜像爆发

## 关键状态 (Buffs & Debuffs)

*   **法术连击 (Hot Streak)** (Buff): ID `48108`。瞬发炎爆术。**核心机制**，4T1强化后触发频率大幅提升。

*   **强化灼烧 (Improved Scorch)** (Debuff): ID `22959`。5% 暴击易伤。(同类效果：术士-暗影烈焰/冬寒刺骨)

## 逻辑分析 (优先级队列)

### 常规单体循环 (Single Target)

1.  **活动炸弹 (Living Bomb)**
    *   **条件**: 目标身上没有活动炸弹，且目标存活时间 > 12秒。
    *   **细节**: 炸弹爆炸是核心伤害来源，且能触发法术连击。**严禁** 在炸弹爆炸前重新施放（会顶掉爆炸伤害）。

2.  **炎爆术 (Pyroblast) - 法术连击**
    *   **条件**: `法术连击` Buff 存在。
    *   **理由**: 免费、瞬发、高伤。是火法输出的核心构件。
    *   **连号处理**: 如果你在读火球时触发了连击，打完火球立刻接炎爆（火球+炎爆几乎同时命中）。

3.  **燃烧 (Combustion)**
    *   **条件**: 冷却就绪，且目标身上有活动炸弹。
    *   **时机**: 尽量配合 4T10 爆发期。但 3.3.5 的燃烧机制是“增加暴击几率直到暴击 3 次”，是一个不稳定的增益，CD 好了就用由于 CD 较短。

4.  **镜像 (Mirror Image) - 4T10 爆发**
    *   **条件**: 冷却就绪，且是 Boss 战/高血量目标。
    *   **理由**: 获得 18% 法伤加成，极强爆发。

5.  **灼烧 (Scorch) - 补易伤**
    *   **条件**: 目标缺少 5% 暴击 Buff (强化灼烧/暗影烈焰)。
    *   **理由**: 5% 暴击对火法至关重要。

6.  **火球术 (Fireball)**
    *   **条件**: 站桩填充。
    *   **理由**: 平稳输出，不仅造成伤害还能叠点燃。

7.  **火冲击 (Fire Blast)**
    *   **条件**: 移动中无法读条。

### AOE 逻辑

1.  **活动炸弹**: 给多目标挂炸弹（利用爆炸伤害）。
2.  **烈焰风暴 (Flamestrike)**: 针对聚集目标 (等级9 + 等级8 轮流使用以叠加地面的 DoT)。
3.  **冲击波 (Blast Wave) / 龙息术 (Dragon's Breath)**: 填充。
4.  **暴风雪 (Blizzard)**: 目标数量极大时。

## 拟定 Lua 代码逻辑 - T1阶段

```lua
-- 1. 最高优先级：法术连击炎爆
if buff(B_HotStreak).up then
    return 2 -- 炎爆术 (瞬发)
end

-- 2. 维持活动炸弹 (不要在爆炸前刷新)
if not debuff(S_LivingBomb).up and target.time_to_die > 12 then
    return 3 -- 活动炸弹
end

-- 3. 爆发：燃烧 (配合炸弹和4T1效果)
if spell(S_Combustion).ready and debuff(S_LivingBomb).up then
    return 5 -- 燃烧
end

-- 4. 维持5%暴击易伤 (如果缺少)
if not debuff(D_ImprovedScorch).up 
   and not debuff(D_ShadowMastery).up 
   and not debuff(D_WintersChill).up 
   and not player.moving then
    return 7 -- 灼烧
end

-- 5. 移动填充
if player.moving then
    if spell(S_FireBlast).ready then return 6 end -- 火冲击
    if buff(B_HotStreak).up then return 2 end -- 移动中瞬发炎爆
    return 7 -- 灼烧
end

-- 6. 主要填充：火球术
return 1 -- 火球术
```

**核心思路**: 
- 4T1强化了法术连击触发率，所以循环重心是**疯狂暴击触发连击**
- 保持炸弹和易伤Debuff，其余时间全力输出火球术等待连击触发
- 燃烧CD好了就用（增加暴击率 = 更多连击）
- 移动时优先消耗连击，其次火冲击/灼烧

## 泰坦重铸特色说明 - T1阶段
*   **4T1核心价值**: 务必获取4件套T1！+2%暴击率看似不多，但对法术连击触发率有约10%的提升，是火法的质变。
*   **高暴击收益**: 随着装等提升（T1阶段装备暴击率约30%~40%），配合4T1效果，法术连击触发频率极高。
*   **连击机制**: 你的循环会变成 `炸弹 -> 火球(暴击) -> 炎爆(瞬发) -> 火球(暴击) -> 炎爆(瞬发)` 的机关枪模式。
*   **点燃叠加**: 频繁的暴击会让点燃DoT持续叠加，形成恐怖的持续伤害。
*   **装备优先级**: 命中17%(达标) > 暴击(堆到极限) > 法伤 > 急速
*   **饰品利用**: 优先选择暴击触发类饰品（如奥金顿徽记），配合燃烧使用。

## 常见问题 (FAQ)

### 为什么不使用霜火箭 (Frostfire Bolt)？
在 3.3.5a 版本中，火法主要有两个流派：**折磨火 (TTW Fire)** 和 **霜火法 (FFB Fire)**。
但是在泰坦时光服（及所有后期高端团队环境）中，**火球术 (Fireball)** 是绝对首选，原因如下：

1.  **伤害成长性 (Scaling)**: 火球术拥有更高的 **法术强度 (SP)** 加成系数。随着泰坦服装备等级的提升，火球术的单发伤害上限远高于霜火箭。
2.  **折磨弱小 (Torment the Weak)**: 火球术天赋（折磨火 20/51/0）可以点出奥术系的 `折磨弱小`，对被减速目标造成 **12% 额外伤害**。在团队本中 BOSS 身上必定有减速效果（DK/战士提供），这 12% 独立增伤是霜火箭无法比拟的。
3.  **命中溢出**: 霜火箭流派的一个主要优势是天赋提供命中。但在泰坦服后期装备下，命中往往已经达标甚至溢出，霜火箭的这个优势不再存在。

**结论**: 除非你处于刚满级且命中严重不足的极端前期，否则请使用 **火球术**。本插件逻辑仅支持火球术流派。

### T1套装的装备搭配建议
由于4T1效果强大，装备选择应围绕套装展开：
1. **命中达标**: 利用2T1的+1%命中，装备只需提供12%-13%（团队Buff后达到17%）
2. **极限暴击**: 剩余装备位全部堆暴击。目标：裸装40%+，Buff后50%+
3. **法伤其次**: 在命中达标、暴击堆满的前提下，选择法伤最高的装备
4. **急速适量**: 不需要刻意堆急速，自然成长即可

**推荐非套装部位**: 选择MC/BWL中暴击+法伤最高的散件（如奥妮克希亚项链、法师之刃等）。
