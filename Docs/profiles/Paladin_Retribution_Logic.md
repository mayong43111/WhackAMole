# 泰坦重铸 惩戒骑士单体输出指南 - T1阶段

> **版本**: WotLK 3.3.5a | **配置文件**: Paladin_Retribution.lua v7 | **优化**: 基于WCL高端玩家日志分析

## 快速配置

**战斗前准备**：
- 🔴 **圣印**: 复仇圣印 (需要31点天赋) | 没有就用命令圣印
- 🔵 **祝福**: 单人用力量祝福 | 团队用王者祝福  
- ⚡ **审判**: 动作条必须放智慧审判 (回蓝核心)

**核心机制**: 基于冷却管理(FCFS)，无圣能系统。卡CD使用高伤技能，审判回蓝维持续航。

## 单体输出循环

### 优化原理 (基于WCL高端日志)

**WCL数据来源**: 拉格纳罗斯25人Normal击杀 (2:26战斗时长)

**关键发现**:
1. **审判无条件卡CD**: CD利用率88%，无蓝量条件限制
2. **清算之手高优先级**: CD利用率96%，几乎卡CD
3. **神圣愤怒不使用**: 高端玩家完全不使用（DPE太低）
4. **驱邪术仅触发时**: 只在战争艺术触发时使用，不强制卡CD
5. **奉献低优先级**: CD利用率66%，说明是空GCD填充

**核心机制**:
1. **圣印叠层**: 复仇圣印需5层DoT (约7-8秒)，满层前不开复仇之怒
2. **蓝量管理**: 审判卡CD持续回蓝，通常不需要切圣印
3. **斩杀阶段**: 愤怒之锤最高优先级

### 技能优先级

**斩杀阶段 (<20%血量)**:
```
1. 复仇之怒 (圣印满5层时)
2. 愤怒之锤 ⭐核心
3. 审判 (无条件卡CD)
4. 十字军打击
5. 清算之手
6. 神圣风暴
```

**常规阶段**:
```
1. 复仇之怒 (圣印满5层时)
2. 审判 (最高优先级，无条件卡CD) ⭐
3. 十字军打击 (卡CD)
4. 清算之手 (几乎卡CD)
5. 神圣风暴
6. 驱邪术 (仅战争艺术触发)
7. 奉献 (低优先级填充)
8. 智慧圣印 (蓝<30%危机)
9. 复仇圣印 (蓝>70%恢复)
```

**与旧版本差异**:
- ✅ 审判提升至第2优先级，移除蓝量条件
- ❌ 移除神圣愤怒（高端玩家不使用）
- ⚠️ 驱邪术仅触发时使用，不强制卡CD

## APL配置代码 (v7)

当前插件使用的SimC风格APL（基于WCL高端日志优化）：

```lua
-- 斩杀阶段 (<20%血量)
"actions+=/avenging_wrath,if=target.health.pct<20&dot.holy_vengeance.stack>4",
"actions+=/hammer_of_wrath,if=target.health.pct<20",
"actions+=/judgement,if=target.health.pct<20",
"actions+=/crusader_strike,if=target.health.pct<20",
"actions+=/hand_of_reckoning,if=target.health.pct<20",
"actions+=/divine_storm,if=target.health.pct<20",

-- 常规阶段
"actions+=/avenging_wrath,if=dot.holy_vengeance.stack>4",  
  -- 复仇之怒：等待复仇圣印叠满5层(>4即为5层)才开启
  -- 理由：满层时DoT伤害最高，最大化爆发期输出

"actions+=/judgement",
  -- 审判：最高优先级，无条件卡CD
  -- 理由：触发"审判贤者"回蓝(15%基础法力) + 高伤害
  -- WCL分析：高端玩家CD利用率88%，无蓝量条件限制
  
"actions+=/crusader_strike",
  -- 十字军打击：4秒CD，核心物理输出
  -- 理由：2T1套装+6%伤害，必须卡CD使用
  -- WCL分析：CD利用率87%

"actions+=/hand_of_reckoning",
  -- 清算之手：8秒CD，高伤害单体技能
  -- 理由：仅在目标未攻击你时造成高伤害
  -- WCL分析：CD利用率96%，几乎卡CD
  
"actions+=/divine_storm",
  -- 神圣风暴：9秒CD(4T1)，高伤AOE技能
  -- 理由：4T1减CD后单体伤害也极高
  -- WCL分析：CD利用率86%
  
"actions+=/exorcism,if=buff.the_art_of_war.up",
  -- 驱邪术：仅战争艺术触发时使用（瞬发）
  -- 理由：高伤法术，触发时优先使用
  -- WCL分析：高端玩家不强制卡CD，仅触发时使用
  
"actions+=/consecration,if=!buff.consecration.up|buff.consecration.remains<2",
  -- 奉献：低优先级填充技能
  -- 理由：持续8秒，CD 8秒，空GCD时使用
  -- WCL分析：CD利用率仅66%，蓝耗高DPE一般
  
"actions+=/seal_of_wisdom,if=mana.percent<30&cooldown.divine_plea.remains>0",               
  -- 神圣恳求：蓝量<70%紧急回蓝
  -- 理由：降低25%治疗但回蓝量大，防止空蓝
  
"actions+=/seal_of_wisdom,if=mana.percent<30&cooldown.divine_plea.remains>0",
  -- 智慧圣印：蓝量<30%且神圣恳求CD中时临时切换
  -- 理由：紧急回蓝手段，DPS会降低但避免空蓝
  
"actions+=/seal_of_vengeance,if=mana.percent>70&!buff.seal_of_vengeance.up",
  -- 复仇圣印：蓝量>70%且当前不是复仇圣印时切回
  -- 理由：蓝量恢复后切回高DPS圣印，但需7-8秒重新叠层
```

**关键条件说明**:
- `dot.holy_vengeance.stack>4`: 复仇圣印5层满层
- `mana.percent<30`: 蓝量危机阈值（紧急切换圣印）
- `buff.the_art_of_war.up`: 战争艺术触发瞬发驱邪术
- `buff.consecration.up/remains<2`: 奉献buff检查（避免覆盖浪费）
- `!buff.seal_of_vengeance.up`: 当前没有复仇圣印buff（可以切回）

**v7版本变更**:
- ✅ 审判无蓝量条件，直接卡CD
- ❌ 移除神圣愤怒和神圣恳求
- ⚠️ 驱邪术只保留触发时使用的规则

**圣印切换策略**:
- 正常战斗：全程保持复仇圣印
- 蓝量危机（<30%）：临时切智慧圣印回蓝
- 蓝量恢复（>70%）：切回复仇圣印（需重新叠5层）
- 切换代价：触发GCD + 复仇圣印重新叠层7-8秒
- **重要**: 审判卡CD通常提供足够回蓝，很少需要切换圣印

## DPS故障排查

**DPS过低检查清单**:

1. ✅ **圣印**: 复仇圣印(联盟)/命令圣印 - 鼠标移到buff图标确认
2. ✅ **祝福**: 力量祝福/王者祝福 - 确认有buff
3. ✅ **审判**: 动作条必须是智慧审判 (ID: 53408)，最高优先级卡CD
4. ✅ **命中/精准**: 命中8% (264等级)、精准26 (214等级)
5. ✅ **卡CD**: 审判>十字军打击>清算之手>神圣风暴，不浪费CD
6. ✅ **圣印层数**: 复仇圣印战斗开始需7-8秒叠满，满层前DPS低属正常
7. ✅ **清算之手**: 必须目标没有看你(非当前T)才造成伤害，确认仇恨列表
8. ✅ **技能槽位**: 8个技能（无神圣愤怒），确保都在动作条上

## 常见问题

**Q: 为什么没有复仇圣印？**  
A: 需要惩戒天赋31点解锁，等级64+，去训练师学习。没有就用命令圣印(DPS低15%)。

**Q: 为什么清算之手也在循环里？**  
A: 在此环境下，清算之手对**没有攻击你的目标**造成的伤害极高(DPE超越十字军打击)。如果你OT了或者你在抗怪，它就没有伤害。

**Q: 为什么审判优先级这么高？**  
A: 审判触发"审判贤者"天赋回蓝(15%基础法力)，是唯一持续回蓝手段，不卡CD会OOM。

**Q: 为什么神圣风暴单体也要打？**  
A: 4T1减CD至9秒，伤害系数高，是仅次于十字军打击的核心输出技能。

**Q: 复仇之怒何时开？**  
A: 等待复仇圣印叠满5层(约7-8秒)，或斩杀阶段(<20%)立即开启。

**Q: 没有圣能系统吗？**  
A: WotLK版本无圣能，完全基于技能CD的FCFS系统(大灾变4.0才引入圣能)。

**Q: v7版本和v5版本有什么区别？**
