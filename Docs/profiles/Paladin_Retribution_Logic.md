# 泰坦重铸 (Titan Forged) 惩戒骑士输出逻辑文档 - T1阶段

## 关于服务器版本 - 泰坦重铸 (Titan Forged)
**确认信息**: 这是一个基于 3.3.5a (巫妖王之怒) 核心的中国区特别版本服务器。
- **技能系统**: 使用完整的巫妖王之怒技能机制（圣印系统、审判、神圣风暴、复仇之怒等）
- **装备系统**: 使用魔改的经典旧世T1套装（当前阶段），套装效果针对WOTLK机制重新设计
本插件逻辑完全遵循 3.3.5a 机制，并针对服务器的**魔改T1套装**特效进行优化。

## 核心机制
3.3.5a 版本的惩戒骑士是一个基于**冷却管理 (FCFS)** 的系统。

### 属性说明 (Attributes)
**关于惩戒骑士的属性**:
*   惩戒骑士在 3.3.5a 是标准的板甲近战DPS，依赖力量和暴击。
*   **输出模式**: **优先使用冷却就绪的高伤害技能**（First Come First Served）。不存在“攒豆/终结”机制。

### 属性优先级 (Stats Priority)
1.  **命中 (Hit)**: 8% (达标) - 物理技能命中
2.  **精准 (Expertise)**: 26 (达标) - 减少被招架和躲闪
3.  **力量 (Strength)**: 核心属性，提供攻击强度和法术强度
4.  **暴击 (Crit)**: 重要属性，触发战争艺术等效果
5.  **急速 (Haste)**: 提升平砍速度和部分技能CD，收益一般
6.  **护甲穿透 (ArP)**: T1阶段装备稀少，收益较低

### 套装特效 - 魔改T1 (无畏套装)
泰坦重铸服务器的 T1 套装针对WOTLK机制重新设计：

*   **2T1**: 【十字军打击】所造成的伤害提高 **6%**
    *   **实战机制**: 纯粹的被动伤害提升，不改变优先级。
    *   **优先级**: 保持十字军打击作为核心输出技能。
    
*   **4T1**: 【神圣风暴】技能的冷却时间缩短 **1秒**
    *   **机制说明**: 
        - **原始CD**: 10秒
        - **4T1后**: 9秒
    *   **实战意义**: 
        - 极大地提高了神圣风暴的覆盖率。
        - **AOE与单体**: 由于神圣风暴本身伤害系数较高（武器伤害%），在CD缩短后，即使是**单体战斗**也应优先或卡CD使用。
    *   **数学模型**: 60秒内可多打出约1次神圣风暴。

## 关键状态 (Buffs & Debuffs)

*   **战争艺术 (The Art of War)** (Buff): ID `59578`
    - 自动攻击或技能暴击触发。
    - 使**驱邪术**瞬发。
    - **策略**: 触发后将其插入技能间隙（不延后物理技能前提下）。

*   **复仇之怒 (Avenging Wrath)** (Buff): ID `31884`
    - 爆发期，通常配合饰品/药水。

## 逻辑分析 (T1 优先级队列)

这是基于 **Hekili Titan** 优化版的逻辑调整：

### 单体/AOE 通用循环 (FCFS)

1.  **愤怒之锤 (Hammer of Wrath)**
    *   **条件**: 目标血量 < 20%
    *   **理由**: 斩杀期伤害最高，必爆。

2.  **审判 (Judgement)** (回蓝核心)
    *   **条件**: CD就绪
    *   **理由**: 惩戒骑唯一的续航手段（审判贤者/智慧审判）。**优先级提至最高**以避免长时间战斗空蓝。

3.  **神圣风暴 (Divine Storm)** (4T1 核心)
    *   **条件**: CD就绪
    *   **理由**: 4T1下CD仅9秒，伤害系数高，且能带出鲜血圣印/命令圣印伤害。AOE和单体皆为核心。

4.  **十字军打击 (Crusader Strike)** (2T1 强化)
    *   **条件**: CD就绪
    *   **理由**: 2T1伤害+6%，CD极短，是填充GCD的主要手段。

5.  **驱邪术 (Exorcism)**
    *   **条件**: 触发**战争艺术** (瞬发)
    *   **理由**: 瞬发高伤法术，优先于奉献。

6.  **奉献 (Consecration)**
    *   **条件**: 目标在范围内
    *   **理由**: 填充技能，耗蓝较高。

7.  **神圣愤怒 (Holy Wrath)**
    *   **条件**: 亡灵/恶魔目标
    *   **理由**: 此类目标下伤害极高，否则优先级垫底。

## APL 配置逻辑 - T1阶段 (SimC 风格)

本插件使用类 SimulationCraft 的 APL (Action Priority List) 进行优先级配置。

```lua
    apl = {
        -- 优先级 1: 复仇之怒 (CD好爆发)
        -- 只要CD转好就使用，最大化输出窗口
        "actions+=/avenging_wrath",

        -- 优先级 2: 愤怒之锤 (斩杀)
        -- 只有在血量低于 20% 或 复仇之怒 期间可用
        "actions+=/hammer_of_wrath,if=target.health.pct<20|buff.avenging_wrath.up",

        -- 优先级 3: 审判 (回蓝 & 伤害)
        -- 触发“审判贤者”回蓝的关键 (提升优先级以防止缺蓝)
        "actions+=/judgement",

        -- 优先级 4: 神圣风暴 (核心伤害 / 4T1)
        -- 4T1 将CD减少至9秒。单体和AOE都是高优先级。
        "actions+=/divine_storm",

        -- 优先级 5: 十字军打击 (核心伤害 / 2T1)
        -- 2T1 增加 6% 伤害。
        "actions+=/crusader_strike",

        -- 优先级 6: 驱邪术 (战争艺术)
        -- 仅在瞬发时使用
        "actions+=/exorcism,if=buff.the_art_of_war.up",

        -- 优先级 7: 奉献 (填充)
        -- 放置地面效果
        "actions+=/consecration,if=!debuff.consecration.up",

        -- 优先级 8: 神圣愤怒 (填充 / AOE)
        "actions+=/holy_wrath",
        
        -- 优先级 9: 制裁之锤 (最后的填充手段)
        -- 可选：仅当没有其他技能可用时
        "actions+=/hammer_of_justice"
    }
```

### 代码逻辑说明

**优先级设计理念**：
1. **爆发优先** - 复仇之怒CD好就用，最大化输出窗口
2. **卡CD技能** - 十字军打击、审判、神圣风暴绝不浪费CD
3. **4T1优化** - 神圣风暴CD缩短后，其优先级提升
4. **触发优先** - 驱邪术（瞬发）触发时应尽快使用
5. **填充伤害** - 奉献、神圣愤怒保持持续输出

**T1套装优化**：
- **2T1**: 十字军打击伤害+6%，强化了核心技能。
- **4T1**: 神圣风暴CD缩短至9秒，无论单体还是多目标，都应高频使用。
- **AOE与单体统一**: 由于神圣风暴的高伤害系数和4T1的减CD，它在单体循环中也占有核心地位。

**关键机制**：
- **无资源限制**: WotLK 惩戒骑基于蓝量和CD，没有豆子（圣能）系统。
- **审判回蓝**: 必须卡CD使用审判来保持续航（审判贤者）。
- **斩杀 (愤怒之锤)**: 仅在斩杀线或开爆发时可用，优先级极高。

## 泰坦重铸特色说明 - T1阶段

*   **4T1核心价值**: 务必获取4件套T1！神圣风暴CD缩短10%，输出循环更紧凑，DPS提升明显。
*   **2T1价值**: 十字军打击+6%伤害，稳定提升单体伤害。
*   **神圣风暴优先**: 获得4T1后，神圣风暴不仅是AOE技能，也是核心单体技能。
*   **装备优先级**: 命中8% > 精准26 > 力量 > 暴击 > 急速 > 护甲穿透（T1阶段破甲装备稀少）
*   **套装优先**: 4T1效果对整体流畅度提升明显，2T1对单体稳定提升，优先凑齐4件套。

## 常见问题 (FAQ)

### 为什么神圣风暴也是单体核心技能？
在这个版本中，神圣风暴造成武器伤害的百分比伤害，且CD适中（特别是4T1后）。它是除了十字军打击外最高效的物理技能，因此单体也必须卡CD打。

### APL 为什么要检测 `buff.avenging_wrath.up` 来使用愤怒之锤？
在WotLK中，通常愤怒之锤只能对20%血量以下目标使用。但在某些特定版本或套装下（以及复仇之怒开启期间），可能允许使用。这里保留检查是为了兼容性，实际上主要靠血量判断。

### 为什么审判现在的优先级很高？
**为了防止空蓝 (OOM)**。我们修正了循环，将审判的优先级提高到了十字军打击之前。因为天赋 [智者审判] (Judgements of the Wise) 需要审判命中才能触发回蓝效果。如果先打十字军打击导致审判延后，可能会导致蓝量断档。并且必须要使用 `Judgement of Wisdom` (ID: 53408) 才能确保持续回蓝。

### 需要关注蓝量吗？
需要。虽然惩戒骑有审判贤者回蓝，但在高急速或长时间战斗中，如果审判被打断或未命中，可能会空蓝。**务必卡CD打审判**。

### 为什么没有圣能（豆子）？
这是 3.3.5a (WotLK) 版本，圣能系统是 4.0 (Cataclysm) 才引入的。当前的循环完全基于技能冷却时间 (FCFS)。
