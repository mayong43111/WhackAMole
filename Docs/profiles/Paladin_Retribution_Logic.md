# 泰坦重铸 惩戒骑士单体输出指南 - T1阶段

> **版本**: WotLK 3.3.5a | **配置文件**: Paladin_Retribution.lua v5 | **优化**: 基于Hekili社区版本

## 快速配置

**战斗前准备**：
- 🔴 **圣印**: 复仇圣印 (需要31点天赋) | 没有就用命令圣印
- 🔵 **祝福**: 单人用力量祝福 | 团队用王者祝福  
- ⚡ **审判**: 动作条必须放智慧审判 (回蓝核心)

**核心机制**: 基于冷却管理(FCFS)，无圣能系统。卡CD使用高伤技能，审判回蓝维持续航。

## 属性与装备

**属性优先级**:
1. 命中 8% (264等级) → 精准 26 (214等级) → 力量 > 暴击 > 急速

**魔改T1套装** (无畏套装):
- **2件套**: 十字军打击伤害 +6%
- **4件套**: 神圣风暴CD -1秒 (10秒→9秒) ⭐核心收益

## 单体输出循环

### 优化原理 (Hekili社区版)

1. **圣印叠层**: 复仇圣印需5层DoT (约7-8秒)，满层前不开复仇之怒
2. **蓝量管理**: 智慧审判<90%回蓝，神圣恳求<70%紧急回蓝
3. **斩杀阶段**: 愤怒之锤最高优先级

### 技能优先级

**斩杀阶段 (<20%血量)**:
```
1. 复仇之怒 (圣印满5层时)
2. 愤怒之锤 ⭐核心
3. 审判 (蓝<90%)
4. 十字军打击
5. 神圣风暴
```

**常规阶段**:
```
1. 复仇之怒 (圣印满5层时)
2. 审判 (蓝<90%优先)
3. 十字军打击
4. 神圣风暴
5. 驱邪术 (战争艺术触发)
6. 奉献 (填充)
7. 神圣愤怒 (填充)
8. 神圣恳求 (蓝<70%紧急)
9. 智慧圣印 (蓝<30%危机)
10. 复仇圣印 (蓝>70%恢复)
```

## APL配置代码 (v5)

当前插件使用的SimC风格APL：

```lua
-- 斩杀阶段 (<20%血量)
"actions+=/avenging_wrath,if=target.health.pct<20&dot.holy_vengeance.stack>4",
"actions+=/hammer_of_wrath,if=target.health.pct<20",
"actions+=/judgement,if=target.health.pct<20&mana.percent<90",
"actions+=/crusader_strike,if=target.health.pct<20",
"actions+=/divine_storm,if=target.health.pct<20",

-- 常规阶段
"actions+=/avenging_wrath,if=dot.holy_vengeance.stack>4",  
  -- 复仇之怒：等待复仇圣印叠满5层(>4即为5层)才开启
  -- 理由：满层时DoT伤害最高，最大化爆发期输出
  
"actions+=/judgement,if=mana.percent<90",                  
  -- 审判：蓝量低于90%时最高优先级
  -- 理由：触发"审判贤者"回蓝(15%基础法力)，防止OOM
  
"actions+=/judgement",                                      
  -- 审判：蓝量充足时也要卡CD使用
  -- 理由：维持智慧审判debuff，持续回蓝 + 伤害
  
"actions+=/crusader_strike",                               
  -- 十字军打击：4秒CD，核心物理输出
  -- 理由：2T1套装+6%伤害，必须卡CD使用
  
"actions+=/divine_storm",                                  
  -- 神圣风暴：9秒CD(4T1)，高伤AOE技能
  -- 理由：4T1减CD后单体伤害也极高，必须卡CD
  
"actions+=/exorcism,if=buff.the_art_of_war.up",           
  -- 驱邪术：战争艺术触发时瞬发
  -- 理由：高伤法术，触发时优先使用(避免覆盖)
  
"actions+=/exorcism",                                      
  -- 驱邪术：没有触发时CD好就用
  -- 理由：15秒CD的高伤法术，填充技能
  
"actions+=/consecration,if=!buff.consecration.up|buff.consecration.remains<2",
  -- 奉献：奉献buff不存在或剩余<2秒时施放
  -- 理由：持续8秒，CD 8秒，避免覆盖浪费DPS和蓝量
  -- 注意：必须检查buff，否则会在buff还在时就释放导致浪费
  
"actions+=/holy_wrath",                                    
  -- 神圣愤怒：对亡灵/恶魔高伤
  -- 理由：填充技能，CD好就用
  
"actions+=/divine_plea,if=mana.percent<70",               
  -- 神圣恳求：蓝量<70%紧急回蓝
  -- 理由：降低25%治疗但回蓝量大，防止空蓝
  
"actions+=/seal_of_wisdom,if=mana.percent<30&cooldown.divine_plea.remains>0",
  -- 智慧圣印：蓝量<30%且神圣恳求CD中时临时切换
  -- 理由：紧急回蓝手段，DPS会降低但避免空蓝
  
"actions+=/seal_of_vengeance,if=mana.percent>70&!buff.seal_of_vengeance.up",
  -- 复仇圣印：蓝量>70%且当前不是复仇圣印时切回
  -- 理由：蓝量恢复后切回高DPS圣印，但需7-8秒重新叠层
```

**关键条件说明**:
- `dot.holy_vengeance.stack>4`: 复仇圣印5层满层
- `mana.percent<90/70/30`: 蓝量管理阈值（审判/恳求/切圣印）
- `buff.the_art_of_war.up`: 战争艺术触发瞬发驱邪术
- `buff.consecration.up/remains<2`: 奉献buff检查（避免覆盖浪费）
- `cooldown.divine_plea.remains>0`: 神圣恳求CD中（需要额外回蓝）
- `!buff.seal_of_vengeance.up`: 当前没有复仇圣印buff（可以切回）

**圣印切换策略**:
- 正常战斗：全程保持复仇圣印
- 蓝量危机（<30%）：临时切智慧圣印回蓝
- 蓝量恢复（>70%）：切回复仇圣印（需重新叠5层）
- 切换代价：触发GCD + 复仇圣印重新叠层7-8秒，不到万不得已不要切换

## DPS故障排查

**DPS过低检查清单**:

1. ✅ **圣印**: 复仇圣印(联盟)/命令圣印 - 鼠标移到buff图标确认
2. ✅ **祝福**: 力量祝福/王者祝福 - 确认有buff
3. ✅ **审判**: 动作条必须是智慧审判 (ID: 53408)
4. ✅ **命中/精准**: 命中8% (264等级)、精准26 (214等级)
5. ✅ **卡CD**: 审判、十字军打击、神圣风暴不浪费CD
6. ✅ **圣印层数**: 复仇圣印战斗开始需7-8秒叠满，满层前DPS低属正常

## 常见问题

**Q: 为什么没有复仇圣印？**  
A: 需要惩戒天赋31点解锁，等级64+，去训练师学习。没有就用命令圣印(DPS低15%)。

**Q: 为什么审判优先级这么高？**  
A: 审判触发"审判贤者"天赋回蓝(15%基础法力)，是唯一持续回蓝手段，不卡CD会OOM。

**Q: 为什么神圣风暴单体也要打？**  
A: 4T1减CD至9秒，伤害系数高，是仅次于十字军打击的核心输出技能。

**Q: 复仇之怒何时开？**  
A: 等待复仇圣印叠满5层(约7-8秒)，或斩杀阶段(<20%)立即开启。

**Q: 没有圣能系统吗？**  
A: WotLK版本无圣能，完全基于技能CD的FCFS系统(大灾变4.0才引入圣能)。
