# 泰坦重铸 (Titan Forged) 惩戒骑士输出逻辑文档 - T1阶段

## 关于服务器版本 - 泰坦重铸 (Titan Forged)
**确认信息**: 这是一个基于 3.3.5a (巫妖王之怒) 核心的中国区特别版本服务器。
- **技能系统**: 使用完整的巫妖王之怒技能机制（圣印系统、审判、神圣风暴、复仇之怒等）
- **装备系统**: 使用魔改的经典旧世T1套装（当前阶段），套装效果针对WOTLK机制重新设计
本插件逻辑完全遵循 3.3.5a 机制，并针对服务器的**魔改T1套装**特效进行优化。

## 核心机制
3.3.5a 版本的惩戒骑士是一个基于**冷却管理**和**圣能优化**的动态系统。

### 属性说明 (Attributes)
**关于惩戒骑士的属性**:
*   惩戒骑士在 3.3.5a 是标准的板甲近战DPS，依赖力量和暴击。
*   **圣能系统**: 惩戒骑士使用圣能（Holy Power）作为资源（需要天赋支持），通过十字军打击等技能生成，神圣风暴等技能消耗。

### 属性优先级 (Stats Priority)
1.  **命中 (Hit)**: 8% (达标) - 对于法术伤害技能如审判、奉献，需要17%法术命中
2.  **精准 (Expertise)**: 26 (达标) - 减少被招架和躲闪
3.  **力量 (Strength)**: 核心属性，提供攻击强度和法术强度
4.  **暴击 (Crit)**: 重要属性，触发审判之怒等效果
5.  **急速 (Haste)**: 提升攻击速度和GCD，收益良好
6.  **护甲穿透 (ArP)**: T1阶段装备稀少，不强求

### 套装特效 - 魔改T1 (无畏套装)
泰坦重铸服务器的 T1 套装针对WOTLK机制重新设计：

*   **2T1**: 【十字军打击】所造成的伤害提高 **6%**
    *   **实战价值**: 十字军打击是惩戒骑士的核心技能，既是主要圣能生成手段，又是重要的直接伤害来源
    *   **频率**: CD 4.5秒（天赋缩减后约3秒），意味着极高的使用频率
    *   **收益**: 6%提升看似不多，但考虑到高频率使用，总体DPS提升约4-5%
    *   **优先级**: 中等。主要为凑4件套的过渡效果
    
*   **4T1**: 【神圣风暴】技能的冷却时间缩短 **1秒**
    *   **机制说明**: 
        - **原始CD**: 神圣风暴基础CD为10秒
        - **4T1后**: CD缩短至9秒，缩短幅度10%
    *   **实战意义**: 
        - **AOE场景**: 多目标战斗中神圣风暴是主要伤害技能，CD缩短意味着更高的AOE输出
        - **圣能管理**: 更频繁地使用神圣风暴可以更有效地避免圣能溢出
        - **治疗增益**: 神圣风暴会治疗附近3名队友，更频繁使用提供更多团队支援
    *   **数学模型**: 60秒战斗中
        - 无套装: 60s ÷ 10s = 6次神圣风暴
        - 4T1后: 60s ÷ 9s ≈ 6.7次神圣风暴 (+11%使用次数)
    *   **优化策略**: 多目标场景优先使用神圣风暴而非圣殿骑士的裁决
    
*   **重点**: 魔改T1专注于强化**核心技能的使用频率和伤害**，2T1强化单体核心（十字军打击），4T1强化AOE核心（神圣风暴）

## 关键状态 (Buffs & Debuffs)

*   **审判之怒 (The Art of War)** (Buff): ID `59578`
    - 自动攻击暴击或某些技能暴击时触发
    - 使下一个驱邪术或闪光术瞬发并提高伤害
    - **核心价值**: 免费的高伤害瞬发技能，移动战利器
    
*   **圣能 (Holy Power)** (Resource): 最大3点
    - 通过十字军打击、审判等技能生成
    - 用于施放圣殿骑士的裁决或神圣风暴
    - **核心价值**: 资源管理的关键，避免溢出
    - **4T1影响**: 神圣风暴CD缩短后，圣能消耗更频繁，溢出风险降低

*   **复仇之怒 (Avenging Wrath)** (Buff): ID `31884`
    - 提升20%伤害，持续20秒
    - **核心价值**: 最强爆发期，配合饰品使用

## 逻辑分析 (优先级队列)

### 常规单体循环

1.  **十字军打击 (Crusader Strike)**
    *   **条件**: CD就绪
    *   **理由**: 生成圣能的核心技能。**2T1后伤害+6%**，使其既是资源生成也是重要伤害来源
    *   **重要性**: 卡CD使用，绝不浪费。2T1强化后优先级更高

2.  **审判 (Judgement)**
    *   **条件**: CD就绪
    *   **理由**: 高伤害远程法术攻击。卡CD使用
    *   **注意**: 确保有活跃的圣印（通常是命令圣印）

3.  **圣殿骑士的裁决 (Templar's Verdict)**
    *   **条件**: 圣能=3点
    *   **理由**: 最高的单体圣能消耗技能
    *   **优先级**: 3点圣能时立即使用，避免溢出

4.  **复仇之怒 (Avenging Wrath)**
    *   **条件**: CD就绪，Boss战
    *   **理由**: 20%伤害提升，大爆发技能
    *   **时机**: 配合饰品和英勇/嗜血使用

5.  **正义之锤 (Hammer of Wrath)**
    *   **条件**: 目标<20%血量，或复仇之怒激活
    *   **理由**: 高伤害斩杀技能，优先级高于奉献

6.  **驱邪术 (Exorcism) - 审判之怒触发**
    *   **条件**: 审判之怒Buff存在
    *   **理由**: 瞬发高伤害，不浪费触发
    *   **移动**: 移动战时优先使用

7.  **奉献 (Consecration)**
    *   **条件**: 地面AOE不存在或即将结束
    *   **理由**: 填充伤害，保持DPS。**2T1后持续时间+2秒**

8.  **神圣愤怒 (Holy Wrath)**
    *   **条件**: CD就绪，面对恶魔/亡灵，或AOE场景
    *   **理由**: 额外伤害技能

### AOE循环

1.  **神圣风暴 (Divine Storm)**: 圣能=3时优先使用。**4T1后CD缩短至9秒**，成为主要AOE输出手段
2.  **十字军打击 (Crusader Strike)**: 生成圣能。**2T1后伤害+6%**，保持高优先级
3.  **奉献 (Consecration)**: 维持地面AOE
4.  **审判 (Judgement)**: 保持触发圣印效果
5.  **神圣愤怒 (Holy Wrath)**: 针对特定目标类型

**4T1优势**: CD缩短后，神圣风暴使用频率提升11%，多目标场景DPS显著提升

## 拟定 Lua 代码逻辑 - T1阶段

```lua
-- 实际APL逻辑（已实装）

-- 1. 复仇之怒 (大爆发 - CD好就用)
if spell(S_AvengingWrath).ready then
    return 4 -- 复仇之怒
end

-- 2. 十字军打击 (圣能生成 - 最高优先级)
-- 2T1效果：伤害提高6%
if spell(S_CrusaderStrike).ready then
    return 1 -- 十字军打击
end

-- 3. 审判 (卡CD使用)
if spell(S_Judgement).ready then
    return 2 -- 审判
end

-- 4. 圣殿骑士的裁决 (圣能消耗)
-- 条件：圣能=3点
if player.holy_power >= 3 and spell(S_TemplarsVerdict).ready then
    return 3 -- 圣殿骑士的裁决
end

-- 5. 正义之锤 (斩杀技能)
-- 条件：目标<20%血量 或 复仇之怒激活
if spell(S_HammerOfWrath).ready and (target.health_pct < 20 or buff(B_AvengingWrath).up) then
    return 9 -- 正义之锤
end

-- 6. 驱邪术 (审判之怒触发时)
if buff(B_TheArtOfWar).up and spell(S_Exorcism).ready then
    return 8 -- 驱邪术
end

-- 7. 奉献 (填充 - 2T1后持续时间+2秒)
if not debuff(D_Consecration).up and spell(S_Consecration).ready then
    return 5 -- 奉献
end

-- 8. 神圣愤怒 (CD技能)
if spell(S_HolyWrath).ready then
    return 7 -- 神圣愤怒
end

-- 9. 神圣风暴 (AOE/圣能消耗)
-- 多目标时优先于圣殿骑士的裁决
if active_enemies >= 2 and player.holy_power >= 3 and spell(S_DivineStorm).ready then
    return 6 -- 神圣风暴
end
```

### 代码逻辑说明

**优先级设计理念**：
1. **爆发优先** - 复仇之怒CD好就用，最大化输出窗口
2. **卡CD技能** - 十字军打击、审判绝不浪费CD
3. **圣能管理** - 3点圣能立即消耗，避免溢出
4. **4T1优化** - 强化裁决Buff触发时立即使用
5. **填充伤害** - 奉献、神圣愤怒保持持续输出

**T1套装优化**：
- **2T1**: 十字军打击伤害+6%，强化了核心圣能生成技能的伤害贡献
- **4T1**: 神圣风暴CD缩短至9秒，多目标场景更倾向于使用神圣风暴消耗圣能
- 十字军打击保持最高优先级（除爆发外），既生成圣能又享受2T1加成
- AOE场景优先级提升，4T1使神圣风暴成为更频繁的圣能消耗选择

**关键机制**：
- 圣能管理：避免在3点圣能时继续使用十字军打击造成溢出
- 审判之怒：移动战时优先消耗瞬发驱邪术
- 正义之锤：斩杀阶段或复仇之怒期间才能使用
- AOE判断：4T1后更倾向于使用神圣风暴（CD短，覆盖广）

## 常见问题 (FAQ)

### 为什么要优先使用十字军打击而不是审判？
十字军打击是圣能生成的核心，且2T1提供6%伤害加成，使其既是资源生成也是重要伤害来源。虽然审判法术伤害更高，但圣能管理才是惩戒骑士的DPS基础。

### 4T1的神圣风暴CD缩短有多重要？
**非常重要**！对于AOE场景，CD从10秒缩短至9秒意味着60秒内可以多使用约1次神圣风暴，整体AOE输出提升约11%。同时更频繁的圣能消耗也降低了资源溢出的风险。

### T1阶段惩戒骑士的定位如何？
T1阶段惩戒骑士是稳定的近战DPS，虽然不如后期那么强势，但4T1的爆发能力使其在Boss战中表现优秀。同时提供光环、祝福等团队增益，是优秀的团队成员。

### 圣能为什么要避免溢出？
圣能最大3点，当圣能已经是3点时，再使用十字军打击生成的圣能会直接浪费。因此要养成3点圣能立即消耗的习惯，最大化资源利用效率。
