# 泰坦重铸 (Titan Forged) 平衡德鲁伊输出逻辑文档 - T1阶段

## 关于服务器版本 - 泰坦重铸 (Titan Forged)
**确认信息**: 这是一个基于 3.3.5a (巫妖王之怒) 核心的中国区特别版本服务器。
- **技能系统**: 使用完整的巫妖王之怒技能机制（日月蚀循环、野性成长、星辰坠落等）
- **装备系统**: 使用魔改的经典旧世T1套装（当前阶段），套装效果针对WOTLK机制重新设计
本插件逻辑完全遵循 3.3.5a 机制，并针对服务器的**魔改T1套装**特效进行优化。

## 核心机制
3.3.5a 版本的平衡德鲁伊是一个基于**日月蚀循环（Eclipse）**的能量平衡系统。

### 属性说明 (Attributes)
**关于平衡德鲁伊的属性**:
*   平衡德鲁伊在 3.3.5a 是标准的远程法系DPS，依赖法术强度和暴击。
*   **日月蚀系统**: 通过施放愤怒（Wrath）和星火（Starfire）积累能量条，交替触发日蚀（Solar Eclipse，+40%奥术/自然伤害）和月蚀（Lunar Eclipse，+40%自然伤害）。

### 属性优先级 (Stats Priority)
1.  **命中 (Hit)**: 17% (达标) - 对于法术伤害，需要17%法术命中
    *   团队配置完善（暗牧/鸟德自带 +3%，德莱尼 +1%）时，装备提供 13%-14% 即可
2.  **法术强度 (Spell Power)**: 核心属性，提供所有法术的基础伤害
3.  **暴击 (Crit)**: 重要属性，提升输出和法力回复（通过梦境之境天赋）
4.  **急速 (Haste)**: 提升施法速度和DoT跳动频率，收益良好
5.  **精神 (Spirit)**: 通过【枭兽形态】天赋转换为法术伤害（约23%转化率）

### 套装特效 - 魔改T1 (雷霆之心战甲)
泰坦重铸服务器的 T1 套装针对WOTLK机制重新设计：

*   **2T1**: 【自然之力】天赋召唤的树人数量提高 **1个**
    *   **机制说明**: 
        - **自然之力 (Force of Nature)**: CD 3分钟，召唤3个树人战斗30秒
        - **2T1后**: 召唤4个树人（+33%树人数量）
    *   **实战价值**: 
        - 树人伤害约占总输出的3-5%，增加1个树人意味着提升约1.5%总体DPS
        - 更多树人提供更多AOE清场能力
    *   **优先级**: 中等。主要为凑4件套的过渡效果
    
*   **4T1**: 【星辰坠落】天赋的持续时间提高 **2秒**
    *   **机制说明**: 
        - **星辰坠落 (Starfall)**: CD 90秒，持续10秒的AOE伤害技能
        - **4T1后**: 持续时间从10秒延长至12秒（+20%持续时间）
    *   **实战意义**: 
        - 星辰坠落是平衡德最强AOE技能，单目标也有可观伤害（约占5-8%总伤害）
        - 持续时间延长20%意味着多2次伤害波次（每秒1次）
        - **数学模型**: 假设星辰坠落基础伤害10万
            - 无套装: 10秒 × 1万/秒 = 10万伤害
            - 4T1后: 12秒 × 1万/秒 = 12万伤害 (+20%星辰坠落伤害)
    *   **优化策略**: 星辰坠落CD好就用，尤其在日蚀/月蚀期间
    
*   **重点**: 魔改T1专注于强化**爆发技能的持续能力**，2T1增强召唤物，4T1延长顶级AOE技能持续时间

## 逻辑分析 (优先级队列)

### 常规单体循环

1.  **星辰坠落 (Starfall)**
    *   **条件**: CD就绪
    *   **理由**: 最强爆发技能，单目标和AOE都有可观伤害。**4T1后持续12秒**，伤害提升20%
    *   **重要性**: CD好就用，不要等待完美时机

2.  **自然之力 (Force of Nature)**
    *   **条件**: CD就绪（3分钟）
    *   **理由**: 召唤树人提供额外伤害。**2T1后召唤4个树人**
    *   **时机**: 开场或爆发期使用，配合英勇/嗜血

3.  **月火术 (Moonfire)**
    *   **条件**: DoT不存在或即将结束（<3秒）
    *   **理由**: 核心DoT之一，占比15-20%总伤害
    *   **注意**: 月蚀期间施放伤害更高

4.  **虫群 (Insect Swarm)**
    *   **条件**: DoT不存在或即将结束（<3秒）
    *   **理由**: 核心DoT之二，占比10-15%总伤害
    *   **优先级**: 仅次于月火术

5.  **星火 (Starfire) - 月蚀期间**
    *   **条件**: 月蚀Buff存在
    *   **理由**: 月蚀期间星火伤害提升40%，是主要输出技能
    *   **目标**: 尽可能在月蚀期间多施放星火

6.  **愤怒 (Wrath) - 日蚀期间**
    *   **条件**: 日蚀Buff存在
    *   **理由**: 日蚀期间愤怒伤害提升40%，是主要输出技能
    *   **目标**: 尽可能在日蚀期间多施放愤怒

7.  **愤怒 (Wrath) - 推日蚀**
    *   **条件**: 无日蚀/月蚀，能量条偏向月蚀侧
    *   **理由**: 通过施放愤怒推动能量条向日蚀方向移动

8.  **星火 (Starfire) - 推月蚀**
    *   **条件**: 无日蚀/月蚀，能量条偏向日蚀侧
    *   **理由**: 通过施放星火推动能量条向月蚀方向移动

### AOE循环

1.  **星辰坠落 (Starfall)**: 顶级AOE，CD好就用。**4T1后持续12秒**
2.  **飓风 (Hurricane)**: 固定站位时的引导型AOE
3.  **月火术 (Moonfire)**: 多目标铺DoT（高价值目标）
4.  **虫群 (Insect Swarm)**: 多目标铺DoT（高价值目标）

### 代码逻辑说明

**优先级设计理念**：
1. **爆发优先** - 星辰坠落、自然之力CD好就用，最大化输出窗口
2. **DoT维持** - 月火术、虫群保持高覆盖率（占比25-35%总伤害）
3. **日月蚀优化** - 在对应日蚀/月蚀Buff期间优先施放对应技能（+40%伤害）
4. **能量平衡** - 无日蚀/月蚀时默认施放愤怒推动日蚀

**T1套装优化**：
- **2T1**: 自然之力树人+1，约提升1.5%总体DPS
- **4T1**: 星辰坠落持续时间+2秒，星辰坠落自身伤害提升20%（约1-1.5%总体DPS）
- DoT维持是核心，月火术+虫群占比25-35%总伤害
- 日蚀/月蚀循环是DPS的关键，尽可能在Buff期间施放对应技能

**关键机制**：
- **日月蚀循环**: 通过施放愤怒和星火交替触发日蚀和月蚀，在Buff期间施放对应技能
- **DoT快照**: 月火术在高法强时施放，DoT伤害会记录施放时的属性
- **移动战**: 利用瞬发DoT（月火术、虫群）进行移动战输出
- **AOE场景**: 星辰坠落是顶级AOE，4T1后持续12秒，覆盖范围广

## 常见问题 (FAQ)

### 为什么要优先使用星辰坠落而不是等待日蚀/月蚀？
星辰坠落CD 90秒，是平衡德最强爆发技能（单目标和AOE都有），CD好就用可以最大化使用次数。虽然在日蚀/月蚀期间施放伤害更高，但等待会浪费CD，整体收益更低。

### 4T1的星辰坠落持续时间延长有多重要？
**非常重要**！对于AOE场景，星辰坠落从10秒延长至12秒意味着多2次伤害波次，整体伤害提升20%。即使是单目标，星辰坠落也占总伤害的5-8%，4T1带来约1-1.5%总体DPS提升。

### T1阶段平衡德鲁伊的定位如何？
T1阶段平衡德是稳定的远程法系DPS，虽然不如后期那么强势（缺少4T10的日蚀强化），但魔改T1的4件套对星辰坠落的强化使其在AOE场景表现优秀。同时提供【奥术光环】和【梦境之境】光环，是优秀的团队增益职业。

### 日月蚀循环如何管理？
日月蚀循环是平衡德的核心机制：
- **推日蚀**: 施放愤怒，能量条向日蚀方向移动，触发日蚀后愤怒伤害+40%
- **推月蚀**: 施放星火，能量条向月蚀方向移动，触发月蚀后星火伤害+40%
- **关键**: 在对应日蚀/月蚀Buff期间尽可能多施放对应技能，最大化40%伤害加成的收益
- **简化**: 可以使用插件（如日月蚀条）辅助追踪能量条状态
