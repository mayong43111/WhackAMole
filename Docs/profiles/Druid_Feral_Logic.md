# 泰坦重铸 (Titan Forged) 野性德鲁伊输出逻辑文档 - T1阶段

## 关于服务器版本 - 泰坦重铸 (Titan Forged)
**确认信息**: 这是一个基于 3.3.5a (巫妖王之怒) 核心的中国区特别版本服务器。
- **技能系统**: 使用完整的巫妖王之怒技能机制（连击点系统、野蛮咆哮、狂暴、清晰预兆等）
- **装备系统**: 使用魔改的经典旧世T1套装（当前阶段），套装效果针对WOTLK机制重新设计
本插件逻辑完全遵循 3.3.5a 机制，并针对服务器的**魔改T1套装**特效进行优化。

## 核心机制
3.3.5a 版本的野性德鲁伊（猫德）是一个基于**能量管理**和**连击点优化**的动态系统。

### 属性说明 (Attributes)
**关于野性德鲁伊的属性**:
*   野性德鲁伊在 3.3.5a 是标准的敏捷近战DPS，依赖敏捷、护甲穿透和暴击。
*   **能量系统**: 猫形态使用能量（Energy，最大100点），每秒恢复10点（受急速影响）。
*   **连击点系统**: 通过技能（割裂、撕碎）积累连击点（最大5点），用于终结技（凶猛撕咬、撕碎、野蛮咆哮）。

### 属性优先级 (Stats Priority)
1.  **命中 (Hit)**: 8% (达标) - 物理近战命中需要8%
    *   团队配置完善（德莱尼 +1%）时，装备提供 7% 即可
2.  **精准 (Expertise)**: 26 (达标) - 减少被招架和躲闪
3.  **敏捷 (Agility)**: 核心属性，提供攻击强度、暴击、护甲
4.  **护甲穿透 (Armor Penetration)**: T1阶段后期开始堆叠，收益随装备提升
5.  **暴击 (Crit)**: 重要属性，触发清晰预兆（Omen of Clarity）等效果
6.  **急速 (Haste)**: 提升攻击速度和能量回复，收益良好
7.  **力量 (Strength)**: 通过天赋转换为攻击强度，但优先级低于敏捷

### 套装特效 - 魔改T1 (雷霆之心战甲)
泰坦重铸服务器的 T1 套装针对WOTLK机制重新设计：

*   **2T1**: 【横扫（熊）】消耗的怒气降低 **5点**，【横扫（豹）】消耗的能量降低 **5点**
    *   **机制说明**: 
        - **横扫（豹）(Swipe Cat)**: 原始消耗50能量，AOE伤害技能
        - **2T1后**: 消耗45能量（-10%能量消耗）
    *   **实战价值**: 
        - 单目标场景：横扫在2-3目标时开始使用，能量节省意味着更高的技能使用频率
        - AOE场景：多目标时横扫是主要输出手段，能量降低显著提升AOE输出
        - **收益**: AOE场景提升约5-8%，单目标场景价值较低
    *   **优先级**: 中等。主要为AOE场景优化和凑4件套的过渡效果
    
*   **4T1**: 【狂暴】天赋的持续时间提高 **3秒**
    *   **机制说明**: 
        - **狂暴 (Berserk)**: CD 3分钟，原始持续15秒
        - **效果**: 使割裂（Mangle）无CD，降低凶猛撕咬（Ferocious Bite）能量消耗
        - **4T1后**: 持续18秒（+20%持续时间）
    *   **实战意义**: 
        - 狂暴是猫德最强爆发期，持续时间延长意味着更多技能输出
        - **数学模型**: 假设狂暴期间每秒造成5000 DPS
            - 无套装: 15秒 × 5000 = 7.5万伤害
            - 4T1后: 18秒 × 5000 = 9万伤害 (+20%狂暴期伤害)
        - 延长3秒意味着约多3次割裂或1-2次终结技
    *   **数学模型（整体DPS）**: 假设战斗300秒
        - 狂暴使用2次（3分钟CD）
        - 无套装: 2次 × 7.5万 = 15万爆发伤害
        - 4T1后: 2次 × 9万 = 18万爆发伤害 (+20%爆发伤害)
        - **整体DPS提升**: 约2-3%
    *   **优化策略**: 狂暴期间疯狂使用割裂和终结技，最大化输出窗口
    
*   **重点**: 魔改T1专注于强化**AOE能力（2T1）**和**爆发窗口（4T1）**，使猫德在多目标和爆发期表现更强

## 逻辑分析 (优先级队列)

### 常规单体循环

1.  **野蛮咆哮 (Savage Roar)**
    *   **条件**: Buff不存在或即将结束（<2秒），且连击点>=1
    *   **理由**: 30%伤害提升是猫德DPS的基础，必须保持100%覆盖率
    *   **重要性**: 最高优先级，即使1点连击点也要刷新（紧急情况）

2.  **狂暴 (Berserk)**
    *   **条件**: CD就绪，猛虎之怒Buff存在（配合使用）
    *   **理由**: 最强爆发期，CD好就用。**4T1后持续18秒**，多3秒输出窗口
    *   **时机**: 配合猛虎之怒和英勇/嗜血使用

3.  **猛虎之怒 (Tiger's Fury)**
    *   **条件**: CD就绪，能量<30（避免溢出）
    *   **理由**: 60能量恢复 + 15%伤害增益，能量管理核心技能
    *   **优先级**: 尽量配合狂暴使用，其他时候保持CD使用

4.  **扫击 (Rake)**
    *   **条件**: DoT不存在或即将结束（<3秒）
    *   **理由**: 重要流血DoT，占总伤害15-20%
    *   **优先级**: 高优先级维持

5.  **撕裂 (Rip)**
    *   **条件**: DoT不存在或即将结束（<3秒），且连击点=5
    *   **理由**: 最高伤害DoT，占总伤害25-30%
    *   **优先级**: 5点连击点时立即刷新

6.  **凶猛撕咬 (Ferocious Bite)**
    *   **条件**: 连击点=5，扫击和撕裂都存在，野蛮咆哮>10秒
    *   **理由**: 终结技，消耗连击点造成高额直接伤害
    *   **能量管理**: 优先使用额外能量（最多30）提升伤害

7.  **割裂 (Mangle)**
    *   **条件**: CD就绪（1秒），连击点<5
    *   **理由**: 核心连击点生成技能，狂暴期间无CD
    *   **优先级**: 生成连击点的主要手段

8.  **撕碎 (Shred) - 背后**
    *   **条件**: 在目标背后，能量>=42（清晰预兆触发时0能量），连击点<5
    *   **理由**: 高伤害连击点生成技能，优于割裂
    *   **注意**: 必须在背后使用，否则用割裂

9.  **横扫（豹）(Swipe) - AOE**
    *   **条件**: 2+目标，能量>=45（2T1后，原始50）
    *   **理由**: AOE伤害技能。**2T1后能量-5**，多目标价值高
    *   **优先级**: 仅在多目标时使用

### AOE循环

1.  **狂暴 (Berserk)**: 爆发期，割裂无CD
2.  **猛虎之怒 (Tiger's Fury)**: 能量恢复+伤害增益
3.  **扫击 (Rake)**: 高价值目标铺DoT
4.  **横扫（豹）(Swipe)**: 主要AOE输出。**2T1后能量45**，使用频率更高
5.  **割裂 (Mangle)**: 多目标铺流血Debuff，狂暴期疯狂使用
6.  **撕碎 (Shred)**: 能量过剩时使用

**2T1优势**: 横扫能量从50降至45，多目标场景使用频率提升10%，AOE DPS显著提升

## 拟定 Lua 代码逻辑 - T1阶段

```lua
-- 实际APL逻辑（已实装）

-- 1. 野蛮咆哮维持（最高优先级 - 30%伤害增益）
if (not buff(B_SavageRoar).up or buff(B_SavageRoar).remains < 2) and combo_points >= 1 then
    return 6 -- 野蛮咆哮
end

-- 2. 狂暴（大爆发 - CD好就用）
-- 4T1效果：持续时间从15秒延长至18秒

    return 8 -- 狂暴
end

-- 3. 猛虎之怒（能量恢复+小爆发）
if spell(S_TigersFury).ready and player.energy < 30 then
    return 9 -- 猛虎之怒
end

-- 4. 扫击维持（重要DoT）
if (not debuff(D_Rake).up or debuff(D_Rake).remains < 3) and spell(S_Rake).ready then
    return 2 -- 扫击
end

-- 5. 撕裂维持（最高伤害DoT - 5点连击点）
if (not debuff(D_Rip).up or debuff(D_Rip).remains < 3) and combo_points >= 5 and spell(S_Rip).ready then
    return 4 -- 撕裂
end

-- 6. 凶猛撕咬（终结技 - 5点连击点）
-- 条件：扫击和撕裂都存在，野蛮咆哮>10秒
if combo_points >= 5 and debuff(D_Rake).up and debuff(D_Rip).up and buff(B_SavageRoar).remains > 10 and spell(S_FerociousBite).ready then
    return 5 -- 凶猛撕咬
end

-- 7. 割裂（连击点生成 - CD 1秒）
if spell(S_Mangle).ready and combo_points < 5 then
    return 1 -- 割裂
end

-- 8. 撕碎（高伤害连击点生成 - 背后）
-- 条件：在背后，能量>=42（或清晰预兆触发）
if spell(S_Shred).ready and combo_points < 5 and player.energy >= 42 then
    return 3 -- 撕碎
end

-- 9. 横扫（豹）（AOE）
-- 2T1效果：能量消耗从50降至45
if active_enemies >= 2 and player.energy >= 45 and spell(S_SwipeCat).ready then
    return 7 -- 横扫（豹）
end
```

### 代码逻辑说明

**优先级设计理念**：
1. **Buff维持** - 野蛮咆哮必须保持100%覆盖（30%伤害增益）
2. **爆发优先** - 狂暴CD好就用（4T1后18秒），配合猛虎之怒
3. **DoT维持** - 扫击、撕裂占总伤害40-50%，必须高覆盖率
4. **连击点管理** - 5点连击点优先终结技（撕裂>凶猛撕咬）
5. **能量优化** - 猛虎之怒在能量<30时使用，避免溢出

**T1套装优化**：
- **2T1**: 横扫能量从50降至45，AOE场景使用频率提升10%
- **4T1**: 狂暴持续18秒，爆发期延长20%，多约3次割裂或1-2次终结技
- 野蛮咆哮是核心Buff，必须保持100%覆盖率
- 撕裂是最高伤害来源（25-30%），5点连击点施放

**关键机制**：
- **野蛮咆哮**: 30%伤害增益，最高优先级维持（即使1点连击点紧急刷新）
- **连击点管理**: 5点时优先撕裂（DoT）> 凶猛撕咬（直接伤害）> 野蛮咆哮（刷新）
- **能量管理**: 猛虎之怒提供60能量，在能量<30时使用避免溢出
- **清晰预兆**: 触发时撕碎免费（0能量），优先使用
- **狂暴期**: 割裂无CD，疯狂使用生成连击点

## 常见问题 (FAQ)

### 为什么野蛮咆哮优先级这么高？
野蛮咆哮提供30%物理伤害增益，是猫德DPS的基础。如果没有维持100%覆盖率，整体DPS会下降20-25%。即使在紧急情况下（如DoT即将结束），也要优先保证野蛮咆哮覆盖。

### 4T1的狂暴延长3秒有多重要？
**非常重要**！狂暴是猫德唯一的大爆发期（CD 3分钟），持续时间从15秒延长至18秒（+20%）意味着：
- 多约3次割裂（狂暴期无CD）
- 多1-2次终结技（凶猛撕咬或撕裂）
- 整体爆发期伤害提升20%，总体DPS提升约2-3%

### 撕裂和凶猛撕咬如何选择？
**优先级**: 撕裂（DoT） > 凶猛撕咬（直接伤害）
- **撕裂**: 22秒DoT，占总伤害25-30%，必须保持覆盖
- **凶猛撕咬**: 直接伤害，用于连击点和能量过剩时消耗
- **原则**: 撕裂DoT即将结束时，优先刷新撕裂；否则使用凶猛撕咬

### 横扫（豹）什么时候使用？
**单目标**: 不使用（能量效率低）
**2目标**: 可以使用，2T1后能量45，勉强可接受
**3+目标**: 主要AOE输出手段，2T1后能量降低，使用频率提升10%

### T1阶段野性德鲁伊的定位如何？
T1阶段猫德是顶级单体物理DPS，虽然不如后期那么强势（缺少4T10的撕裂暴击强化），但魔改T1的4件套对狂暴的强化使其爆发期表现优秀。同时通过【强化割裂】为团队提供流血伤害增益，是优秀的近战DPS职业。
