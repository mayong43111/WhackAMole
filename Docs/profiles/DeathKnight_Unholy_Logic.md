# 泰坦重铸 邪恶死亡骑士单体输出指南 - T1阶段

> **版本**: WotLK 3.3.5a | **配置文件**: DeathKnight_Unholy.lua v1 | **优化**: 基于Hekili社区版本

## 快速配置

**战斗前准备**：
- 🔴 **脸**: 鲜血灵气 (单体DPS最高) | 邪恶灵气 (跑位/AOE)
- 🧟 **宠物**: 必须召唤食尸鬼 (永久存在)
- 🎺 **号角**: 保持寒冬号角buff

**核心机制**: 疾病维持(双病) + 天灾打击(主要输出) + 缠绕(符能消耗) + 枯萎凋零(卡CD)。

## 属性与装备

**属性优先级**:
1. 命中 8% (263等级) → 精准 26 (214等级)
2. 力量 > 急速 > 暴击 > 护甲穿透

**魔改T1套装** (无畏套装):
- **2件套**: 寒冬号角产生的符文能量都提高 5点
- **4件套**: 凋零缠绕和冰霜打击造成的伤害提高 5%
    - 邪DK主要收益在于**凋零缠绕**的伤害提升

## 单体输出循环

### 优化原理

1. **疾病快照**: 起手触发高AP/急速buff后刷新疾病，可以提高疾病全程伤害(特别是石像鬼阶段)
2. **枯萎凋零**: 在3.3.5版本，邪DK的枯萎凋零单体伤害极高，必须卡CD使用
3. **荒凉覆盖**: 鲜血打击提供的<荒凉>(伤害+5%)buff不能断
4. **宠物管理**: 保持食尸鬼狂乱(如果天赋点了)

### 技能优先级

**爆发/起手**:
```
1. 凋零缠绕 (开怪前)
2. 冰冷触摸 (上病)
3. 暗影打击 (上病)
4. 枯萎凋零
5. 鲜血打击 (触发荒凉 + 转死亡符文)
6. 鲜血打击 (转死亡符文)
7. 召唤石像鬼 (触发饰品/十字军后)
8. 符文武器增效 (空资源爆发/刷新符文)
```

**常规循环**:
```
1. 冰冷触摸/暗影打击 (如果疾病断了)
2. 召唤石像鬼 (有CD且触发Buff时)
3. 枯萎凋零 (卡CD，单体核心)
4. 鲜血打击 (若符文是鲜血且没有<荒凉>buff)
5. 天灾打击 (消耗邪+冰/死亡符文)
6. 鲜血打击 (转死亡符文)
7. 凋零缠绕 (符能>80 或 触发骤亡)
8. 骨疽 (自动攻击/技能触发，被动伤害)
```

## APL配置代码 (SimC风格)

```lua
-- 疾病管理
"actions+=/icy_touch,if=!dot.frost_fever.ticking",
"actions+=/plague_strike,if=!dot.blood_plague.ticking",

-- 爆发
"actions+=/summon_gargoyle,if=buff.potion_of_speed.up|buff.bloodlust.up|time<20",
-- 这是一个简单的石像鬼逻辑，实际可能需要更复杂的判定

-- 核心循环
"actions+=/death_and_decay,if=cooldown.death_and_decay.ready", -- 卡CD放凋零
"actions+=/scourge_strike,if=runes.unholy>=1&runes.frost>=1", -- 天灾打击
"actions+=/blood_strike,if=buff.desolation.remains<3|runes.blood>=1", -- 维持荒凉或消耗鲜血符文

-- 符能消耗
"actions+=/death_coil,if=runic_power>80|buff.sudden_doom.react", -- 防溢出或免费缠绕

-- 填充
"actions+=/horn_of_winter",
```

**关键机制说明**:
- `buff.desolation`: 邪恶天赋核心buff，增加5%伤害，由鲜血打击触发，持续20秒。
- `Summon Gargoyle`: 天地双鬼是邪DK爆发的标志，石像鬼不仅享受面板属性，还能吃到嗜血等极速加成。

## 常见问题

**Q: 为什么要卡CD放枯萎凋零(DnD)？**  
A: 在WotLK 3.3.5版本中，枯萎凋零的总伤害/符文比（DPE）远高于天灾打击，即使是单体目标也是如此。

**Q: 为什么日志中骨疽伤害很高？**  
A: 骨疽是邪DK的核心被动天赋，它附加在自动攻击和部分技能上。高急速（如工程手套、嗜血、狂乱）和高AP会显著放大骨疽的伤害占比。

**Q: 关于鲜血打击和枯萎凋零的顺序？**  
A: 日志显示起手快速打出枯萎凋零优先级极高。为了最大化石像鬼期间的符文利用，通常会先打枯萎凋零，再用鲜血打击转符文，最后开启石像鬼。

**Q: 为什么不用心脏打击？**  
A: 邪DK没有心脏打击天赋（那是血DK的），且鲜血打击是为了转死亡符文服务于天灾打击。

**Q: 食尸鬼狂乱需要手动补吗？**  
A: 如果天赋点了<食尸鬼狂乱>，通常建议写宏绑在某些技能上，或者由插件自动监控保持，提供宠物急速。
