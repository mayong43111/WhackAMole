## 核心机制
敏锐贼在泰坦服T1阶段可拆分为两个层级：
1. **稳定层（常驻维护）**: 维持切割、割裂、死亡印记，并以出血作为主要构建技能。
2. **爆发层（窗口输出）**: 在高能量（70~80+）时进入暗影之舞，通过伏击/刺骨快速转化连击点。

在此基础上，插件建议拆分为两种可切换版本：
- **版本A：团队增益版（破甲优先）**：维持目标破甲Debuff，为物理队提供稳定收益，同时保证自身输出不塌。
- **版本B：纯输出版（个人DPS优先）**：不主动维护破甲，将全部终结资源用于割裂/刺骨与爆发窗口。

### 泰坦服敏锐实战特征（按NGA样本提炼）

#### 1. 起手强调“资源预装填”
- 常见做法为开怪前准备转嫁星与目标星，追求高星起手（如5星/10星概念）。
- 插件侧不依赖“偷药+翻页宏”实现，而是提供可复现的技能优先级建议。

#### 2. 稳定期以“出血主轴 + 维护体系”为核心
- **构建技能**: 出血
- **维护技能（纯输出）**: 切割、割裂、死亡印记
- **维护技能（团队增益）**: 切割、破甲、割裂、死亡印记
- **终结技能**: 刺骨（在连击点合适且维护稳定时）

#### 3. 爆发窗口是DPS分水岭
- 暗影之舞开启前建议抬高能量（70~80+），防止窗口空转。
- 影舞窗口内优先伏击与刺骨的高频转化。
- 消失可视作额外爆发入口；伺机待发偏向斩杀阶段或关键时间轴。

## 逻辑分析 (优先级队列)

### 插件设计目标

本插件提供**稳定的敏锐循环建议**，不包括：
- ❌ 速度药水、工程炸弹、手套等道具时间轴
- ❌ 一键翻页宏与装备切换宏
- ❌ 团队指定时间轴（如固定易伤窗口）

插件聚焦于：
- ✅ 连击点/能量管理
- ✅ 常驻维护优先级
- ✅ 爆发窗口前的蓄能与窗口内技能顺序
- ✅ 移动与断档时的最小损失选择

### 版本A：团队增益循环（优先维持破甲）

**优先级队列**:
```
1. 切割 - 剩余时间过低时立即补
2. 破甲 - 目标debuff即将结束时优先补
3. 死亡印记（泰坦服） - 可用即保持覆盖
4. 割裂 - 剩余时间过低时补
5. 暗影之舞 - CD就绪且能量>=70时开启
6. 伏击 - 在影舞/潜行窗口中优先
7. 刺骨 - 5星且维护稳定时消耗
8. 出血 - 主构建技能，维持循环
9. 暗影步 - 作为影舞联动或位移补位
```

**核心思路**:
- 先保证“切割+破甲+核心维护”不断，再谈终结伤害。
- 影舞不是“亮了就按”，而是“高能量开窗”。
- 稳定期减少空GCD，默认用出血填充。

### 版本B：纯输出循环（个人DPS优先）

**优先级队列**:
```
1. 切割 - 剩余时间过低时立即补
2. 死亡印记（泰坦服） - 可用即保持覆盖
3. 割裂 - 剩余时间过低时补
4. 暗影之舞 - CD就绪且能量>=70时开启
5. 伏击 - 在影舞/潜行窗口中优先
6. 刺骨 - 5星且三维护稳定时消耗
7. 出血 - 主构建技能，维持循环
8. 暗影步 - 作为影舞联动或位移补位
```

**核心思路**:
- 不主动打破甲，把终结资源集中给割裂/刺骨和爆发窗口。
- 若团队没有其他稳定破甲来源，可临时切回版本A。

### 爆发窗口循环（影舞/消失）

**优先级队列**:
```
1. 爆发前蓄能到70~80+
2. 开暗影之舞（或消失进入潜行窗口）
3. 伏击
4. 刺骨（满足连击点）
5. 伏击
6. 刺骨
7. 窗口结束后立即回到三维护+出血主轴
```

**核心思路**:
- 爆发前检查切割/割裂剩余时间，避免窗口内被迫补维护。
- 伺机待发建议用于斩杀期或关键转火前，提升第二轮爆发完整度。

### 官方环境下的实战趋势（按样本归纳）

**核心设计理念**: 敏锐贼由“高操作爆发职业”向“可模板化执行”的方向演进。

**关键趋势**:
- 常驻维护可模板化（纯输出三维护 / 团队增益四维护），便于插件化监控。
- 爆发窗口密度高，DPS上限取决于窗口质量而非单次技能暴击。
- 宏生态发达，但插件应优先提供可验证的优先级，而非硬编码宏串。

### 实战建议

1. **先稳再爆**: 当前版本维护项不断档是第一优先级。
2. **高能量进窗**: 影舞前抬能量到70~80，收益显著高于“见CD就开”。
3. **斩杀留资源**: 伺机待发和消失尽量与斩杀阶段对齐。
4. **减少过度自动化依赖**: 宏可降操作门槛，但会弱化应对机制变化的灵活性。

## 战斗技巧

### 基础循环

**站桩输出**:
- 切割、割裂、死亡印记优先保持
- 非爆发窗口以出血构建连击点
- 满足条件后用刺骨转化伤害

**触发响应**:
- 暗影之舞接近就绪（约10秒）时开始抬能量
- 影舞开启后优先伏击与刺骨交替

**移动处理**:
- 优先保持维护不断
- 借助暗影步完成贴身与GCD衔接
- 避免在移动中丢失影舞窗口

### 常见错误

❌ **低能量开影舞**: 窗口空转，伤害断层明显
❌ **只顾打刺骨**: 导致切割/割裂断档
❌ **把宏序列等同循环逻辑**: 宏可失误容错，但不等于最优优先级
❌ **爆发技能囤太久**: 除时间轴要求外，不应长期压CD
