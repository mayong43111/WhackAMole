# WhackAMole (打地鼠)

WhackAMole 是一个魔兽世界 (3.3.5a) 插件，旨在通过可视化下一个最佳施法技能来帮助玩家进行循环输出，就像打地鼠游戏一样。它专门针对 Titan-Forged 私服环境进行了优化，但也遵循标准的 WLK 逻辑。

## 设计理念

**本插件专为新手玩家快速学习职业输出循环而设计。**

- 🎯 **学习导向**：帮助新手理解和掌握正确的技能优先级和输出手法。
- ✅ **稳定输出**：保障基本的输出流程正确，避免常见的技能使用错误。
- ⚖️ **合理定位**：不追求极限 DPS，也不会让你垫底——提供稳健的中上水平输出。
- 📚 **培训工具**：通过实战练习逐步建立肌肉记忆，最终脱离辅助独立操作。

如果你是追求极限 DPS 的资深玩家，建议使用 WeakAuras 等高级工具自行定制。本插件更适合想要快速上手并打出合格输出的玩家。

## 功能特性

- **可视化循环辅助**：直接在屏幕上显示下一个需要按下的技能。
- **动态布局**：根据优先级和类型自动排列技能图标。
- **多职业支持**：支持法师、战士、圣骑士、死亡骑士、德鲁伊等多个职业的配置文件。
- **APL 引擎**：基于 SimC 风格的 APL（Action Priority List）系统，灵活配置技能优先级。
- **泰坦服优化**：包含针对自定义服务器机制（如 T10 套装奖励）的逻辑调整。
- **简易配置**：支持导入/导出配置，可视化编辑 APL 脚本。

## 安装说明

1.  下载本仓库。
2.  将 \src\ 文件夹重命名为 \WhackAMole\，然后复制到您的魔兽世界插件目录：
    \Interface\AddOns\WhackAMole\
3.  (可选) 开发者可使用 \publish.ps1\ 脚本自动发布到游戏目录。

## 使用方法

-   输入 **/wam**：打开配置菜单。
-   **右键点击** 插件框体标题栏：访问快速选项（锁定/解锁，清空技能条）。
-   **拖拽** 图标离开栏位：从建议列表中移除该技能（仅限解锁状态）。
-   **拖拽** 法术书中的技能到栏位：手动将技能分配到该位置（战斗中无法操作）。

## 开发者指南 (Developer Guide)

### 项目结构
```
src/
├── Core/           # 核心逻辑、事件处理、配置管理
├── Engine/         # APL 执行器、状态机、SimC 解析器
├── UI/             # 界面绘制与交互（Grid、Options）
├── Classes/        # 职业专属逻辑（战士、法师、圣骑士等）
├── Presets/        # 内置 APL 配置文件
├── Libs/           # 第三方库（LibStub、LibCustomGlow 等）
└── WhackAMole.toc  # 插件加载清单
```

### 开发工具
- 使用 `publish.ps1` 脚本可自动将代码发布到 WoW 插件目录进行测试。
- 参考 `PoCs/` 目录中的概念验证项目了解特定 API 用法。
- 查看 `Docs/` 目录中的设计文档了解架构细节。
